# Competitor Comparison & Alternative Page Guide

Based on research from [Siege Media](https://www.siegemedia.com/seo/programmatic-seo), [G2](https://www.g2.com/), [Ahrefs](https://ahrefs.com/blog/comparison-pages/).

---

## Page Types Overview

### 1. "[Product] Alternatives" (Singular)
**Pattern:** Best [Competitor] Alternatives
**Example:** "Best Notion Alternatives"
**Intent:** User knows what they don't want

### 2. "[Product] Alternatives" (Plural)
**Pattern:** [Competitor] Alternatives & Competitors
**Example:** "HubSpot Alternatives & Competitors"
**Intent:** Exploring the market

### 3. "[A] vs [B]" (Direct Comparison)
**Pattern:** [Your Product] vs [Competitor]
**Example:** "Asana vs Monday"
**Intent:** Decision stage, comparing specific options

### 4. "[A] vs [B] vs [C]"
**Pattern:** [Product A] vs [Product B] vs [Product C]
**Example:** "Slack vs Teams vs Discord"
**Intent:** Broader comparison shopping

## Alternative Page Structure

### Template Structure
```
H1: X Best [Competitor] Alternatives for [Year]

Introduction (100-200 words)
- Acknowledge competitor's strengths
- Explain why people look for alternatives
- Brief overview of what you'll cover

Quick Comparison Table
- Key features at a glance
- Pricing comparison
- Rating summary

Detailed Reviews (per alternative)
- H2: [Alternative Name] - Best for [Use Case]
  - Overview
  - Key Features
  - Pricing
  - Pros/Cons
  - User Reviews Summary
  - Best For

Your Product Section (if relevant)
- How you compare
- Why consider switching

FAQ Section
- Common questions about switching

Conclusion
- Summary recommendation
- CTA
```

## Content Requirements

### Research Checklist
- [ ] Current pricing (check actual pricing pages)
- [ ] Feature lists (complete and accurate)
- [ ] G2/Capterra ratings and review counts
- [ ] Pros/cons from real user reviews
- [ ] Integration lists
- [ ] Company info (founded, location, size)
- [ ] Recent updates/changes

### Data Sources
| Data Type | Sources |
|-----------|---------|
| Features | Product pages, docs |
| Pricing | Pricing pages |
| Reviews | G2, Capterra, TrustRadius |
| Ratings | G2, Capterra, Product Hunt |
| Usage stats | Company press, reports |
| Comparisons | Existing comparison sites |

### Avoiding Bias
- **Acknowledge competitor strengths**
- Use objective criteria
- Include user review quotes
- Show data sources
- Update regularly
- Disclose affiliations

## SEO Optimization

### On-Page Elements
| Element | Optimization |
|---------|-------------|
| Title | [A] vs [B]: [Modifier] [Year] Comparison |
| Meta | Objective, mentions both, includes year |
| H1 | Matches search intent exactly |
| H2s | Feature categories, use cases |
| Images | Product screenshots, comparison graphics |
| Schema | Product, FAQPage, BreadcrumbList |

### Content Depth
| Page Type | Minimum Words | Target |
|-----------|---------------|--------|
| Alternative page | 1500 | 2500-4000 |
| Comparison page | 1200 | 2000-3000 |
| Multi-comparison | 2000 | 3000-5000 |

### Internal Linking
- Link to individual product pages
- Link to related comparisons
- Link to category/roundup pages
- Link from product pages to comparisons

## Conversion Optimization

### CTA Placement
| Location | CTA Type |
|----------|----------|
| After intro | Soft - "See our comparison" |
| After each product | Trial links |
| Your product section | Strong - "Start free trial" |
| After comparison table | "Try [Your Product]" |
| Conclusion | Final recommendation |

### CTA Examples
```
For alternative pages:
"Try [Your Product] free for 14 days →"
"See how [Your Product] compares →"

For comparison pages:
"[Winner] vs [Your Product] - See the comparison →"
"Start your free trial →"
```

### Trust Elements
- Last updated date
- Author with credentials
- Methodology disclosure
- User review quotes
- Rating badges

## Updating Strategy

### Update Triggers
| Change | Action |
|--------|--------|
| Pricing change | Update within 1 week |
| New feature launch | Add to comparison |
| Product pivot | Major content update |
| New competitor | Add to alternative list |
| Rating change | Update quarterly |

### Update Schedule
- **Pricing/features:** Quarterly audit
- **Year in title:** January update
- **New competitors:** As discovered
- **User reviews:** Quarterly refresh
- **Screenshots:** Annually

## Common Mistakes

| Mistake | Impact | Fix |
|---------|--------|-----|
| Outdated pricing | Trust loss | Regular audits |
| Obvious bias | Credibility loss | Objective criteria |
| No screenshots | Low engagement | Add visuals |
| Missing schema | No rich results | Add Product schema |
| No update date | Stale perception | Show "Updated [date]" |
| Thin content | Poor rankings | Add depth, research |
| Wrong intent | Poor rankings | Match search intent |
