## Paywall Types & When to Use

### Hard Paywall
**Definition:** Content/features blocked entirely
**Best for:**
- Premium content with clear value
- Features with high perceived value
- When you have strong brand/demand

**Conversion:** Higher conversion, lower volume

### Soft Paywall (Metered)
**Definition:** Limited free access, then paywall
**Best for:**
- Content sites (X articles/month)
- Usage-based products
- Building habit before conversion

**Conversion:** Lower conversion, higher volume

### Freemium Paywall
**Definition:** Core free forever, premium features locked
**Best for:**
- Products with viral/network effects
- Clear free vs paid differentiation
- High user volume needed

### Feature Gating
**Definition:** Specific features trigger upgrade
**Best for:**
- Power user features
- Team/collaboration features
- Advanced functionality

## Trial Extension & Recovery

### Trial Expiring Sequence
| Timing | Message |
|--------|---------|
| 3 days before | "Your trial ends in 3 days - complete setup" |
| 1 day before | "Last day! Don't lose your work" |
| Expiration day | "Your trial has ended - upgrade to continue" |
| Day after | "Miss us? Here's 7 more days free" |

### Grace Period Strategy
- Extend trial if user showed engagement
- Offer discount for immediate conversion
- Preserve data but limit access
- Re-engagement email sequence

### Win-Back Offers
| Timing | Offer |
|--------|-------|
| Day 1 post-expiry | Extended trial (7 days) |
| Day 3 | First month discount (20-30%) |
| Day 7 | Larger discount (40-50%) |
| Day 30 | Annual discount offer |

## Pricing Psychology

### Anchoring Techniques
1. **Show highest tier first** - Anchors expectations
2. **Cross out original price** - Shows savings
3. **Annual savings displayed** - "Save $120/year"
4. **Compare to alternatives** - "Less than a coffee/day"

### Decoy Effect
```
Basic     Pro (Popular)     Enterprise
$9/mo     $29/mo            $99/mo
5 seats   25 seats          Unlimited
1 GB      25 GB             100 GB
```
Middle tier looks best compared to extremes

## A/B Testing Framework

### High-Impact Tests
| Element | Test Variation | Expected Impact |
|---------|---------------|-----------------|
| Annual default | Annual vs Monthly | +20-40% revenue |
| Social proof | With vs without | +10-20% |
| Feature framing | Benefits vs features | +10-25% |
| Price anchoring | High-first vs low-first | +10-30% |
| Urgency elements | Timer vs no timer | +15-30% |
| Trial length | 7 vs 14 vs 30 days | Varies |

### Testing Priority
1. **Annual vs monthly default** - Revenue impact
2. **Value proposition copy** - Conversion
3. **Price anchoring** - Perceived value
4. **Social proof elements** - Trust
5. **Trigger timing** - When to show
6. **Visual design** - Layout optimization
