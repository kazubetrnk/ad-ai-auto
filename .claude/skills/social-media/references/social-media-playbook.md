## Engagement Strategy

### Engagement Actions
| Action | Frequency | Purpose |
|--------|-----------|---------|
| Reply to comments | Within 1 hour | Build community |
| Comment on others | 10-20/day | Visibility |
| Share UGC | Weekly | Social proof |
| Respond to DMs | Same day | Relationships |
| Like comments | Always | Acknowledgment |

### Community Building
| Tactic | Platform |
|--------|----------|
| Create group | Facebook, LinkedIn |
| Host Spaces | Twitter/X |
| Go Live | Instagram, TikTok |
| Run challenges | TikTok, Instagram |
| Feature followers | All platforms |
| Ask questions | All platforms |

### Handling Negative Comments
| Type | Response |
|------|----------|
| Constructive | Thank, address publicly |
| Confused | Clarify, provide info |
| Trolling | Ignore or hide |
| Legitimate complaint | Move to DM |
| Spam | Delete, block |

|--------|---------|--------|

### Planning Workflow
```
Weekly:
1. Review previous week performance
2. Identify top performers to repeat
3. Plan 7-day content calendar
4. Create/source visual assets
5. Write copy and schedule
6. Prepare engagement time blocks
```

## Crisis Management

### Response Levels
| Level | Situation | Response |
|-------|-----------|----------|
| Low | Minor complaint | Standard response |
| Medium | Multiple complaints | Escalate internally |
| High | Viral negative | Executive response |
| Crisis | Major incident | War room, PR |

### Crisis Response Steps
1. Pause scheduled content
2. Assess situation severity
3. Gather facts internally
4. Draft response (legal review if needed)
5. Respond publicly
6. Monitor and engage
7. Post-mortem and process update
