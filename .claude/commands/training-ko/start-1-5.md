## Claude를 위한 지침

품질 보증과 다각도 피드백을 위해 리뷰어 에이전트를 활용하는 방법을 학생들에게 가르치세요.

### 수업 개요

---

**모듈 1.5: 리뷰어 에이전트 심화 학습**

마케팅 키트에는 6개의 전문 리뷰어 에이전트가 포함되어 있습니다. 이들은 콘텐츠 품질을 보장하기 위해 다양한 관점에서 전문가 수준의 피드백을 제공합니다.

**소요 시간:** 약 30분

---

### 1단계: 리뷰어 에이전트가 필요한 이유는?

모든 마케팅 콘텐츠는 여러 각도에서 검토되어야 합니다:
- 브랜드 일관성
- SEO 최적화
- 전환 잠재력
- 페르소나 적합성
- 실제 구현 가능성

리뷰어 에이전트는 각 영역에 대한 전문가를 온디맨드로 제공합니다.

### 2단계: 전문가 리뷰어

세 가지 전문가 리뷰어를 설명하세요:

**`brand-voice-guardian`:**
- 브랜드 보이스 일관성 보장
- 톤과 메시징 확인
- 브랜드 가이드라인 대조 검증
- 브랜드에 맞지 않는 콘텐츠 표시

**`seo-specialist`:**
- 키워드 사용 평가
- 온페이지 SEO 요소 확인
- 콘텐츠 구조 검토
- 최적화 기회 식별

**`conversion-optimizer`:**
- CTA 효과성 평가
- 설득 요소 검토
- 마찰 지점 식별
- 전환 개선 제안

### 3단계: 페르소나 리뷰어

세 가지 페르소나 리뷰어를 설명하세요:

**`manager-maria` (마케팅 매니저, 38세):**
- B2B 중견 기업 관점
- 팀 구현 적합성 평가
- 리소스 제약 고려
- 캠페인 실현 가능성 검토

**`solo-steve` (1인 사업가, 32세):**
- 1인 비즈니스 관점
- 시간 효율성 평가
- 셀프 서비스 용이성 검토
- 예산 민감도 고려

**`startup-sam` (스타트업 창업자, 28세):**
- 초기 단계 스타트업 관점
- 성장 잠재력 평가
- 바이럴 요소 검토
- 시장 출시 속도 평가

### 4단계: 전문가 리뷰어 활용

전문가 리뷰어로 실습하세요:

```
전문가 리뷰어로 AgentKits 블로그 포스트를 검토하세요:

1. `brand-voice-guardian`: 브랜드 일관성 평가
2. `seo-specialist`: SEO 최적화 평가
3. `conversion-optimizer`: 전환 잠재력 평가

각 리뷰어는 점수(1-10)를 매기고 다음을 제공해야 합니다:
- 상위 3가지 강점
- 상위 3가지 개선 필요 사항
- 구체적인 편집 권장 사항
```

### 5단계: 페르소나 리뷰어 활용

페르소나 리뷰어로 실습하세요:

```
페르소나 관점에서 AgentKits 이메일 시퀀스를 검토하세요:

1. `manager-maria`: B2B 마케팅 매니저가 이것을 가치 있다고 생각할까요?
2. `solo-steve`: 1인 사업가가 이것을 구현할까요?
3. `startup-sam`: 이것이 스타트업의 성장을 촉진할까요?

각 리뷰어는 다음을 평가해야 합니다:
- 그들의 니즈와의 공감도
- 구현 실현 가능성
- 가치 인식

```

### 6단계: 종합 리뷰 보드

포괄적인 피드백을 위해 모든 리뷰어를 함께 사용하세요:

```
AgentKits 랜딩 페이지 카피에 대해 전체 리뷰 보드를 실행하세요:

전문가 리뷰:
- 브랜드 일관성 (brand-voice-guardian)
- SEO 최적화 (seo-specialist)
- 전환 잠재력 (conversion-optimizer)

페르소나 리뷰:
- B2B 적합성 (manager-maria)
- 셀프 서비스 잠재력 (solo-steve)
- 성장 잠재력 (startup-sam)

종합: 모든 리뷰어를 만족시킬 수 있는 변경 사항은 무엇입니까?
```

### 7단계: 반복적 개선

검토-개선 사이클을 보여주세요:

```
1. 콘텐츠 생성
2. 다중 리뷰어 피드백 받기
3. 최우선 변경 사항 구현
4. 변경된 콘텐츠 재검토
5. 모든 리뷰어가 8점 이상을 줄 때까지 반복
```

### 8단계: 어떤 리뷰어를 언제 사용할지

가이드를 제공하세요:

| 콘텐츠 유형 | 주요 리뷰어 |
|-------------|-------------------|
| 블로그 포스트 | seo-specialist, brand-voice-guardian |
| 랜딩 페이지 | conversion-optimizer, manager-maria |
| 이메일 시퀀스 | email-wizard, startup-sam |
| 소셜 콘텐츠 | brand-voice-guardian, solo-steve |
| 영업 자료 | sales-enabler, manager-maria |
