## Claude를 위한 지침

학생이 첫 번째 실제 마케팅 작업을 수행하도록 안내하세요 - 명령어를 사용하여 브랜드 가이드라인을 만드는 것입니다.

### 레슨 개요

---

**모듈 0.2: 첫 번째 마케팅 작업**

이제 실제 마케팅 작업을 해봅시다. `/brand:voice` 명령어를 사용하여 AgentKits의 브랜드 가이드라인을 만들어 보겠습니다.

---

### 1단계: 브랜드 가이드라인 설명

브랜드 가이드라인이 중요한 이유를 설명하세요:
- 모든 콘텐츠의 일관성 보장
- Claude(및 사람)가 올바른 톤으로 작성할 수 있도록 지원
- 핵심 메시지와 용어 문서화
- 브랜드에 맞지 않는 콘텐츠 방지

### 2단계: 브랜드 보이스 명령어 사용

실제 시스템 명령어를 사용하도록 안내하세요:

```
/brand:voice "AgentKits - B2B team productivity coordination tool for remote teams"
```

Claude가 포괄적인 브랜드 가이드라인을 생성하도록 한 다음, 학생과 함께 검토하세요.

### 3단계: 고객 페르소나 생성

이제 페르소나를 위한 리서치 명령어를 사용하세요:

```
/research:persona "Remote team managers at tech companies using AgentKits"
```

설명하세요:
- `researcher` 에이전트가 시장 조사를 처리합니다
- 페르소나는 특정 대상에게 콘텐츠를 타겟팅하는 데 도움이 됩니다
- 이 페르소나들을 코스 전반에 걸쳐 사용할 것입니다

### 4단계: 생성된 결과물 검토

출력물을 함께 검토하세요:

```
Show me what the brand:voice command created
```

다음을 강조하세요:
- 보이스 속성과 톤 스펙트럼
- 메시징 프레임워크
- 사용할 단어와 피해야 할 단어
- 다른 에이전트와의 통합

### 5단계: 컨텍스트의 힘

설명하세요:
- 이러한 가이드라인은 이제 프로젝트 컨텍스트에 존재합니다
- 향후 작업에서 에이전트가 이를 참조할 수 있습니다
- 이것이 "컨텍스트 인식"입니다 - Claude의 슈퍼파워 중 하나입니다
- 코스 전반에 걸쳐 이것을 사용할 것입니다

### 빠른 실습: 콘텐츠 생성 테스트

브랜드 컨텍스트를 사용하여 시도해 보도록 하세요:

```
/content:social "Remote team productivity tips" "linkedin"
```

Claude가 콘텐츠 생성 시 브랜드 컨텍스트를 자동으로 사용하는 방법을 보여주세요.

### 6단계: 다른 주요 명령어 탐색

학생이 마스터할 다른 명령어를 간략하게 시연하세요:

**캠페인 기획:**
```
/campaign:plan "Q1 Product Launch"
```

**SEO 리서치:**
```
/seo:keywords "remote team productivity"
```

**이메일 시퀀스:**
```
/sequence:welcome "AgentKits" "trial users"
```
