### 레슨 개요

---

**모듈 2.6: SEO 최적화**

SEO는 지속 가능한 장기 성장을 견인합니다. 키워드 리서치, 콘텐츠 최적화, 기술적 SEO를 마스터해봅시다.

**소요 시간:** ~40분

---

### 1단계: 키워드 리서치

키워드 리서치 명령어 사용:

```
/seo:keywords "remote team productivity"
```

```
/seo:keywords "team focus time software"
```

```
/seo:keywords "coordination tools for remote teams"
```

키워드 기회 검토:
- 검색량 추정치
- 경쟁 수준
- 검색 의도 분류
- 키워드 클러스터

### 2단계: 경쟁사 SEO 분석

경쟁사 SEO 분석:

```
/seo:competitor "rescuetime.com"
```

```
/seo:competitor "freedom.to"
```

파악 사항:
- 경쟁사가 순위를 차지한 키워드
- 콘텐츠 격차
- 백링크 기회
- 기술적 강점/약점

### 3단계: SEO 감사

종합 감사 실행:

```
/seo:audit "agentkits.com"
```

감사 영역 검토:
- 기술적 SEO (크롤링 가능성, 속도, 모바일)
- 온페이지 SEO (제목, 메타, 헤더)
- 콘텐츠 품질 (깊이, 관련성)
- 백링크 프로필

### 4단계: 콘텐츠 최적화

기존 콘텐츠 최적화:

```
/seo:optimize "blog post about team productivity" "remote team productivity"
```

최적화 항목:
- 타이틀 태그 개선
- 메타 설명
- 헤더 구조 (H1, H2, H3)
- 키워드 배치
- 내부 링크 제안
- 이미지 alt 텍스트

### 5단계: SEO 콘텐츠 생성

SEO 중심의 블로그 명령어 사용:

```
/content:blog "The Complete Guide to Remote Team Productivity" "remote team productivity"
```

그런 다음 최적화:

```
/seo:optimize "the blog post" "remote team productivity"
```

### 6단계: 추천 스니펫 전략

추천 스니펫 최적화:

```
추천 스니펫에 최적화된 콘텐츠 생성:

타겟 쿼리: "How to improve remote team productivity"

단락 스니펫 (40-60 단어):
직접 답변 형식

목록 스니펫 (번호 단계):
단계별 형식

표 스니펫:
비교 형식
```

### 7단계: 기술적 SEO 체크리스트

SEO 체크리스트 사용:

```
/checklist:seo-weekly "agentkits.com"
```

주간 SEO 작업:
- 크롤링 오류 확인
- 페이지 속도 모니터링
- 모바일 사용성
- 색인 상태
- 키워드 추적

### 8단계: 필라-클러스터 전략

필라 콘텐츠 계획 생성:

```
필라-클러스터 SEO 전략 생성:

필라 페이지: "The Complete Guide to Remote Team Productivity"
- 3000-5000 단어, 포괄적

클러스터 콘텐츠 (10개 포스트):
1. "How to Schedule Team Focus Time" → team focus time
2. "Reducing Meetings Without Losing Alignment" → reduce meetings remote
3. "Async Communication Best Practices" → async communication
4. "Deep Work for Remote Teams" → deep work teams
5. "Team Productivity Metrics That Matter" → productivity metrics
...

내부 링크:
- 각 클러스터는 필라로 링크
- 필라는 모든 클러스터로 링크
- 관련 클러스터 간 교차 링크
```

### 9단계: SEO를 고려한 콘텐츠 캘린더

SEO 중심 캘린더 생성:

```
/campaign:calendar "12 weeks - SEO content strategy - target keywords: remote team productivity, team focus time, coordination software"
```

콘텐츠를 키워드에 매핑:
- 1주차: [키워드] - [제목] - [유형]
- 2주차: [키워드] - [제목] - [유형]
...

### 10단계: SEO 측정

SEO 성과 추적:

```
/analytics:report "SEO performance" "organic traffic, keyword rankings, conversions"
```

주요 지표:
- 자연 트래픽 추세
- 키워드 순위 변화
- 자연 전환율
- 페이지 성과
