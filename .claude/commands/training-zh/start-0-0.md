- 核心营销: `attraction-specialist`、`lead-qualifier`、`email-wizard`、`sales-enabler`、`continuity-specialist`、`upsell-maximizer`

**76 个斜杠命令**,分为以下类别:
- `/campaign:*` - 营销活动规划和分析

- `/seo:*` - SEO 优化
- `/analytics:*` - 性能报告
- `/sales:*` - 销售赋能
- `/crm:*` - CRM 和生命周期管理

**营销技能:**

- `paid-advertising`、`content-strategy`、`analytics-attribution`、`brand-building`
