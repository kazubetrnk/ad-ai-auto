## Claude 指令

教导学生利用审查者代理进行质量保证和多角度反馈。

### 课程概述

---

**模块 1.5:审查者代理深度解析**

营销工具包包含 6 个专业审查者代理。它们从不同角度提供专家级反馈,确保内容质量。

**时长:**约 30 分钟

---

### 步骤 1:为什么需要审查者代理?

每一份营销内容都应该从多个角度进行审查:
- 品牌一致性
- SEO 优化
- 转化潜力
- 人物角色匹配度
- 实际可执行性

审查者代理为您提供每个领域的按需专家。

### 步骤 2:专家审查者

解释三个专家审查者:

**`brand-voice-guardian`:**
- 确保品牌声音一致性
- 检查语气和信息传递
- 根据品牌指南进行验证
- 标记偏离品牌的内容

**`seo-specialist`:**
- 评估关键词使用
- 检查页面 SEO 元素
- 审查内容结构
- 识别优化机会

**`conversion-optimizer`:**
- 评估 CTA 效果
- 审查说服元素
- 识别摩擦点
- 建议转化改进

### 步骤 3:人物角色审查者

解释三个人物角色审查者:

**`manager-maria` (营销经理,38 岁):**
- B2B 中型公司视角
- 评估团队执行适配性
- 考虑资源约束
- 审查活动可行性

**`solo-steve` (个体创业者,32 岁):**
- 单人企业视角
- 评估时间效率
- 审查自助服务便利性
- 考虑预算敏感性

**`startup-sam` (创业公司创始人,28 岁):**
- 早期创业公司视角
- 评估增长潜力
- 审查病毒式传播因素
- 评估上市速度

### 步骤 4:使用专家审查者

与专家审查者进行练习:

```
使用专家审查者审查我们的 AgentKits 博客文章:

1. `brand-voice-guardian`:评估品牌一致性
2. `seo-specialist`:评估 SEO 优化
3. `conversion-optimizer`:评估转化潜力

每个审查者应该打分(1-10)并提供:
- 前 3 项优势
- 需要改进的前 3 项
- 具体的编辑建议
```

### 步骤 5:使用人物角色审查者

与人物角色审查者进行练习:

```
从人物角色视角审查我们的 AgentKits 邮件序列:

1. `manager-maria`:B2B 营销经理会觉得这有价值吗?
2. `solo-steve`:个体创业者会实施这个吗?
3. `startup-sam`:这会为创业公司带来增长吗?

每个审查者应该评估:
- 与其需求的共鸣度
- 实施可行性
- 价值感知
```

### 步骤 6:综合审查委员会

将所有审查者结合使用以获得全面反馈:

```
对 AgentKits 落地页文案进行完整的审查委员会评审:

专家审查:
- 品牌一致性 (brand-voice-guardian)
- SEO 优化 (seo-specialist)
- 转化潜力 (conversion-optimizer)

人物角色审查:
- B2B 适配性 (manager-maria)
- 自助服务潜力 (solo-steve)
- 增长潜力 (startup-sam)

综合:哪些改变能满足所有审查者?
```

### 步骤 7:迭代改进

展示审查-改进循环:

```
1. 创建内容
2. 获取多审查者反馈
3. 实施最优先的改进
4. 重新审查修改后的内容
5. 重复直到所有审查者评分 8+ 以上
```

### 步骤 8:何时使用哪个审查者

提供指导:

| 内容类型 | 主要审查者 |
|-------------|-------------------|
| 博客文章 | seo-specialist, brand-voice-guardian |
| 落地页 | conversion-optimizer, manager-maria |
| 邮件序列 | email-wizard, startup-sam |
| 社交内容 | brand-voice-guardian, solo-steve |
| 销售资料 | sales-enabler, manager-maria |
