### 步骤 2:命令类别参考

查看所有命令类别:

| 类别 | 命令 | 使用场景 |
|----------|----------|-----------|
| `/campaign:*` | plan, brief, analyze, calendar | 营销活动管理 |
| `/content:*` | blog, social, email, landing, ads | 内容创作 |
| `/seo:*` | keywords, audit, optimize, competitor | SEO 优化 |
| `/analytics:*` | roi, funnel, report | 性能跟踪 |
| `/sales:*` | pitch, outreach, battlecard, qualify | 销售赋能 |
| `/leads:*` | score, nurture, qualify | 潜在客户管理 |
| `/crm:*` | sequence, segment, score, lifecycle | CRM 自动化 |
| `/sequence:*` | welcome, nurture, re-engage | 邮件序列 |
| `/social:*` | engage, viral, schedule | 社交媒体 |
| `/research:*` | market, persona, trend | 市场研究 |
| `/brand:*` | voice, book, assets | 品牌管理 |
| `/ops:*` | daily, weekly, monthly | 运营 |
| `/report:*` | weekly, monthly | 报告 |
| `/checklist:*` | campaign-launch, seo-weekly | 检查清单 |

### 步骤 5:快速参考命令

分享基本命令快捷方式:

**快速内容:**
```
/content:fast "quick headline for social post"
```

**优质内容:**
```
/content:good "detailed blog post with research"
```

**头脑风暴:**
```
/brainstorm "marketing channel ideas for B2B SaaS"
```

### 步骤 6:运营命令

练习运营工作流程:

**日常运营:**
```
/ops:daily "content creation"
```

**周度回顾:**
```
/ops:weekly "current week dates"
```

**营销活动检查清单:**
```
/checklist:campaign-launch "Q1 Product Launch" "next week"
```

### 步骤 7:完整命令工作流程

演示完整工作流程:

```
1. /research:market "B2B productivity software"
2. /campaign:plan "Q2 Growth Campaign"
3. /seo:keywords "team productivity"
4. /content:blog "productivity topic" "target keyword"
5. /seo:optimize "the blog post" "keyword"
6. /content:social "blog topic" "linkedin"
7. /sequence:nurture "product" "blog readers"
8. /analytics:report "Q2" "all channels"
```
