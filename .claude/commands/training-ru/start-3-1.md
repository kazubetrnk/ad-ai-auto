# Модуль 3, Урок 1: Основы CRO

## Добро пожаловать в оптимизацию конверсии

В этом модуле вы освоите новые навыки CRO (Conversion Rate Optimization - оптимизация коэффициента конверсии), добавленные в AgentKits Marketing. Эти навыки помогут вам систематически улучшать показатели конверсии во всех маркетинговых активах.

## Цели обучения

К концу этого урока вы:
1. Поймете 6 категорий навыков CRO
2. Узнаете, когда использовать каждую команду CRO
3. Примените психологические принципы к конверсиям
4. Создадите свой первый аудит CRO

---

## Набор навыков CRO

AgentKits Marketing включает 7 специализированных навыков CRO:

| Навык | Использовать для | Команда |
|-------|------------------|---------|
| `page-cro` | Целевые страницы, главные страницы, тарифы | `/cro:page` |
| `form-cro` | Формы захвата лидов, контактные формы, формы демо | `/cro:form` |
| `popup-cro` | Модальные окна, наложения, намерение выхода | `/cro:popup` |
| `signup-flow-cro` | Регистрация, пробные подписки | `/cro:signup` |
| `onboarding-cro` | Активация после регистрации | `/cro:onboarding` |
| `paywall-upgrade-cro` | Платные стены в приложении, обновления | `/cro:paywall` |
| `ab-test-setup` | Разработка экспериментов | `/test:ab-setup` |

## Фреймворк CRO

Каждый анализ CRO следует этой иерархии:

### 1. Ценностное предложение (наивысшее влияние)
- Могут ли посетители понять, что вы предлагаете, за 5 секунд?
- Понятна ли выгода, а не только функции?

### 2. Эффективность заголовка
- Передает ли он основную ценность?
- Конкретен и достоверен ли он?

### 3. Оптимизация CTA
- Одно четкое основное действие?
- Выше сгиба, видимо, убедительно?

### 4. Сигналы доверия
- Социальное доказательство рядом с решениями?
- Значки безопасности видны?

### 5. Снижение трения
- Минимальное количество полей формы?
- Четкие следующие шаги?

## Упражнение 1: Аудит целевой страницы AgentKits

Давайте применим принципы CRO к целевой странице AgentKits.

### Текущее состояние (гипотетическое)

**Заголовок:** "Team Productivity Software"
**CTA:** "Learn More"
**Форма:** 7 полей

### Ваша задача

Создайте аудит CRO, используя команду `/cro:page`:

```bash
/cro:page "Analyze AgentKits's landing page: Headline 'Team Productivity Software', CTA 'Learn More', 7-field form. Target: remote team managers."
```

### Ожидаемый результат

Аудит должен выявить:
- Общий заголовок (не конкретный и не ориентированный на выгоду)
- Слабый CTA ("Learn More" вместо ориентированного на действие)
- Высокое трение (7 полей - это слишком много)

## Упражнение 2: Применение психологии

Навык `marketing-psychology` включает более 70 ментальных моделей. Попробуйте:

```bash
/marketing:psychology "How can we use loss aversion and social proof to improve AgentKits's trial signup rate?"
```

### Ключевые модели для CRO

| Модель | Применение |
|--------|------------|
| Боязнь потери | Формулировка "Не упустите" |
| Социальное доказательство | "Присоединяйтесь к 10 000+ команд" |
| Якорение | Показывайте сначала дорогой тариф |
| Дефицит | Ограниченные места или время |
| Взаимность | Бесплатная ценность перед запросом |

## Практическое задание

Создайте полный план улучшения CRO:

1. **Проведите аудит страницы:**
   ```bash
   /cro:page "AgentKits homepage audit"
   ```

2. **Оптимизируйте форму:**
   ```bash
   /cro:form "Reduce AgentKits's 7-field trial signup form"
   ```

3. **Разработайте A/B-тест:**
   ```bash
   /test:ab-setup "Test new headline vs current for AgentKits"
   ```

Сохраните свою работу в:
```
training/exercises/markit/cro/
├── landing-page-audit.md
├── form-optimization.md
└── ab-test-plan.md
```
