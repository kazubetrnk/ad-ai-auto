## Инструкции для Claude

Проведите студента через его первую настоящую маркетинговую задачу - создание руководства по бренду с использованием наших команд.

### Обзор урока

---

**Модуль 0.2: Ваша первая маркетинговая задача**

Теперь давайте займёмся настоящей маркетинговой работой. Мы создадим руководство по бренду для AgentKits, используя команду `/brand:voice`.

---

### Шаг 1: Объясните руководство по бренду

Объясните, почему руководство по бренду важно:
- Обеспечивает согласованность во всём контенте
- Помогает Claude (и людям) писать правильным тоном
- Документирует ключевые сообщения и терминологию
- Предотвращает создание контента, не соответствующего бренду

### Шаг 2: Используйте команду Brand Voice

Направьте их на использование реальной системной команды:

```
/brand:voice "AgentKits - B2B team productivity coordination tool for remote teams"
```

Пусть Claude создаст всестороннее руководство по бренду, затем просмотрите его вместе со студентом.

### Шаг 3: Создайте портреты клиентов

Теперь используйте команду исследования для создания портретов:

```
/research:persona "Remote team managers at tech companies using AgentKits"
```

Объясните:
- Агент `researcher` занимается исследованием рынка
- Портреты помогают нацелить контент на конкретную аудиторию
- Мы будем использовать эти портреты на протяжении всего курса

### Шаг 4: Просмотрите созданное

Просмотрите результаты вместе:

```
Show me what the brand:voice command created
```

Обратите внимание на:
- Атрибуты голоса и спектр тональности
- Структура сообщений
- Слова, которые следует использовать и избегать
- Интеграция с другими агентами

### Шаг 5: Сила контекста

Объясните:
- Эти руководства теперь существуют в контексте проекта
- В будущих задачах агенты смогут ссылаться на них
- Это "осведомлённость о контексте" - одна из суперспособностей Claude
- Мы будем использовать их на протяжении всего курса

### Быстрое упражнение: Тестирование генерации контента

Попросите их попробовать использовать контекст бренда:

```
/content:social "Remote team productivity tips" "linkedin"
```

Покажите, как Claude автоматически использует контекст бренда при создании контента.

### Шаг 6: Изучите другие ключевые команды

Кратко продемонстрируйте другие команды, которые они освоят:

**Планирование кампании:**
```
/campaign:plan "Q1 Product Launch"
```

**SEO исследование:**
```
/seo:keywords "remote team productivity"
```

**Последовательности электронных писем:**
```
/sequence:welcome "AgentKits" "trial users"
```

## Ключевые моменты обучения
- Руководство по бренду обеспечивает согласованность
- Используйте `/brand:voice` для создания руководства по голосу бренда
- Используйте `/research:persona` для портретов клиентов
- Осведомлённость о контексте означает, что агенты ссылаются на существующую работу
- Реальные команды (`/campaign:*`, `/content:*`, `/seo:*`) - это то, что они освоят
