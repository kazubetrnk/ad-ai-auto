SEO обеспечивает устойчивый долгосрочный рост. Давайте освоим исследование ключевых слов, оптимизацию контента и технический SEO.

**Продолжительность:** ~40 минут

---

### Шаг 1: Исследование ключевых слов

Используйте команду исследования ключевых слов:

```
/seo:keywords "remote team productivity"
```

```
/seo:keywords "team focus time software"
```

```
/seo:keywords "coordination tools for remote teams"
```

Изучите возможности ключевых слов:
- Оценки объема поиска
- Уровни конкуренции
- Классификация намерений
- Кластеры ключевых слов

### Шаг 2: Анализ SEO конкурентов

Проанализируйте SEO конкурентов:

```
/seo:competitor "rescuetime.com"
```

```
/seo:competitor "freedom.to"
```

Определите:
- Ключевые слова, по которым они ранжируются
- Пробелы в контенте
- Возможности для обратных ссылок
- Технические сильные/слабые стороны

### Шаг 3: SEO-аудит

Проведите комплексный аудит:

```
/seo:audit "agentkits.com"
```

Проверьте области аудита:
- Технический SEO (сканируемость, скорость, мобильная версия)
- Внутристраничный SEO (заголовки, мета, заголовки)
- Качество контента (глубина, релевантность)
- Профиль обратных ссылок

### Шаг 4: Оптимизация контента

Оптимизируйте существующий контент:

```
/seo:optimize "blog post about team productivity" "remote team productivity"
```

Оптимизация включает:
- Улучшение тега заголовка
- Мета-описание
- Структура заголовков (H1, H2, H3)
- Размещение ключевых слов
- Предложения по внутренним ссылкам
- Альтернативный текст изображений

### Шаг 5: Создание SEO-контента

Используйте команду блога с фокусом на SEO:

```
/content:blog "The Complete Guide to Remote Team Productivity" "remote team productivity"
```

Затем оптимизируйте:

```
/seo:optimize "the blog post" "remote team productivity"
```

### Шаг 6: Стратегия избранных фрагментов

Оптимизируйте для избранных фрагментов:

```
Create content optimized for featured snippets:

Target query: "How to improve remote team productivity"

PARAGRAPH SNIPPET (40-60 words):
Direct answer format

LIST SNIPPET (numbered steps):
Step-by-step format

TABLE SNIPPET:
Comparison format
```

### Шаг 7: Чек-лист технического SEO

Используйте SEO чек-лист:

```
/checklist:seo-weekly "agentkits.com"
```

Еженедельные SEO-задачи:
- Проверка ошибок сканирования
- Мониторинг скорости страницы
- Мобильная юзабилити
- Статус индексации
- Отслеживание ключевых слов

### Шаг 8: Стратегия столпов и кластеров

Создайте план столпового контента:

```
Create pillar-cluster SEO strategy:

PILLAR PAGE: "The Complete Guide to Remote Team Productivity"
- 3000-5000 words, comprehensive

CLUSTER CONTENT (10 posts):
1. "How to Schedule Team Focus Time" → team focus time
2. "Reducing Meetings Without Losing Alignment" → reduce meetings remote
3. "Async Communication Best Practices" → async communication
4. "Deep Work for Remote Teams" → deep work teams
5. "Team Productivity Metrics That Matter" → productivity metrics
...

INTERNAL LINKING:
- Each cluster links to pillar
- Pillar links to all clusters
- Cross-links between related clusters
```

### Шаг 9: Контент-календарь с SEO

Создайте календарь с фокусом на SEO:

```
/campaign:calendar "12 weeks - SEO content strategy - target keywords: remote team productivity, team focus time, coordination software"
```

Сопоставьте контент с ключевыми словами:
- Неделя 1: [Ключевое слово] - [Название] - [Тип]
- Неделя 2: [Ключевое слово] - [Название] - [Тип]
...

### Шаг 10: Измерение SEO

Отслеживайте SEO-показатели:

```
/analytics:report "SEO performance" "organic traffic, keyword rankings, conversions"
```

Ключевые метрики:
- Тренд органического трафика
- Изменения позиций ключевых слов
- Коэффициент органической конверсии
- Производительность страниц
