### Обзор урока

---

**Модуль 2.2: Разработка контент-стратегии**

Контент-стратегия превращает хаотичное создание контента в систематический двигатель роста. Давайте создадим её для AgentKits.

**Продолжительность:** ~40 минут

---

### Шаг 1: Исследовательская основа

Начните с исследования рынка и аудитории:

```
/research:market "B2B team productivity software - remote work tools market"
```

```
/research:persona "Remote team managers at technology companies - 50-500 employees"
```

```
/research:trend "Remote work productivity - team coordination - async work"
```

### Шаг 2: SEO-исследование ключевых слов

Используйте SEO-команды для создания основы ключевых слов:

```
/seo:keywords "remote team productivity"
```

```
/seo:keywords "team coordination software"
```

```
/seo:keywords "deep work for teams"
```

Сгруппируйте ключевые слова в тематические кластеры:
- Кластер 1: Продуктивность удалённых команд
- Кластер 2: Время фокусировки команды
- Кластер 3: Координация без совещаний

### Шаг 3: Анализ контента конкурентов

Проанализируйте контент конкурентов:

```
/seo:competitor "rescuetime.com"
```

Определите:
- Какие темы они охватывают
- Пробелы в контенте, которые мы можем заполнить
- Ключевые слова, по которым они ранжируются

### Шаг 4: Создание контент-календаря

Используйте календарь кампаний для планирования контента:

```
/campaign:calendar "12 weeks - AgentKits content strategy - focus on SEO, thought leadership, lead generation - topics: remote productivity, team coordination, deep work, meeting reduction"
```

### Шаг 5: Определение типов контента

Планируйте контент по этапам воронки:

**TOFU (Осведомлённость):**
- Статьи в блоге (с фокусом на SEO)
- Контент для социальных сетей
- Экспертный контент

**MOFU (Рассмотрение):**
- Сравнительные руководства
- Обучающий контент
- Кейс-стади

**BOFU (Решение):**
- Демонстрации продукта
- Калькуляторы ROI
- Отзывы клиентов

### Шаг 6: Создание стратегии основного контента

Спланируйте стратегию основных страниц:

```
/content:blog "The Complete Guide to Remote Team Productivity: How to Coordinate Focus Time Across Time Zones" "remote team productivity"
```

Кластерный контент (ссылки на основную страницу):
1. Как планировать время фокусировки команды
2. Сокращение совещаний без потери согласованности
3. Лучшие практики асинхронной коммуникации
4. Глубокая работа в удалённых командах
5. Отслеживание продуктивности команд

### Шаг 7: Рабочий процесс создания контента

Используйте команды для контента для каждого элемента:

**Создание статьи в блоге:**
```
1. /seo:keywords "topic" - Исследование ключевых слов
2. /content:blog "title" "keyword" - Создание статьи
3. /seo:optimize "post" "keyword" - Оптимизация
4. Review with seo-specialist agent
5. Review with brand-voice-guardian agent
```

**Создание контента для соцсетей:**
```
1. /content:social "topic" "linkedin" - LinkedIn post
2. /content:social "topic" "twitter" - Twitter thread
3. Review with conversion-optimizer agent
```

### Шаг 8: Интеграция с email

Создайте email-последовательности для взращивания потребителей контента:

```
/sequence:nurture "AgentKits" "blog readers who downloaded guide"
```

### Шаг 9: План распространения контента

Используйте команды для социальных сетей для распространения:

```
/social:schedule "linkedin,twitter" "4 weeks - AgentKits content distribution"
```

### Шаг 10: Система измерения

Настройте аналитику контента:

```
/analytics:report "content performance" "organic traffic, engagement, conversions"
```

Ключевые метрики для отслеживания:
- Органический трафик по каждому элементу контента
- Время на странице
- Коэффициент конверсии для каждого контента
- Качество лидов из контента

## Ключевые моменты обучения
- Стратегия превращает контент из хаотичного в систематический
- Команды `/research:*` создают основу
- `/seo:keywords` выявляет возможности
- Основная страница + кластер = мощь SEO
- Производство контента следует повторяемому рабочему процессу
- Измерение обеспечивает подотчётность
