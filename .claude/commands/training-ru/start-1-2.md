---

**Модуль 1.2: Работа с маркетинговыми файлами**

Как маркетолог, вы работаете со многими типами ресурсов: брифами кампаний, черновиками контента, исследовательскими документами, аналитическими отчетами. Давайте научимся эффективно организовывать их и управлять ими.

**Продолжительность:** ~25 минут

---

### Шаг 1: Изучение структуры документации

Покажите им папку docs:

```
List all files in docs/
```

Объясните каждый файл документации:
- `brand-guidelines.md` - Шаблон стандартов бренда
- `content-style-guide.md` - Стандарты написания, призывы к действию, форматирование
- `campaign-playbooks.md` - Проверенные шаблоны кампаний
- `channel-strategies.md` - Тактики для конкретных платформ
- `analytics-setup.md` - Отслеживание и атрибуция
- `usage-guide.md` - Полный справочник системы

### Шаг 2: Изучение сценариев кампаний

Прочитайте сценарии кампаний:

```
Read docs/campaign-playbooks.md
```

Объясните типы сценариев:
- Сценарий запуска продукта
- Сценарий генерации лидов
- Сценарий повышения узнаваемости бренда
- Сценарий удержания
- Сценарий продвижения мероприятий

### Шаг 3: Практика команд для контента

Проведите их через команды создания контента:

**Статья в блоге:**
```
/content:blog "5 Ways Remote Teams Can Improve Coordination" "remote team productivity"
```

**Контент для социальных сетей:**
```
/content:social "Team coordination tips for remote managers" "linkedin"
```

**Текст для email:**
```
/content:email "welcome" "trial users for AgentKits"
```

### Шаг 4: Практика команд поиска

Обучите техникам поиска с помощью grep/find или запросов к Claude:

```
Find all files that mention "lead scoring"
```

```
Search for files containing "conversion rate"
```

### Шаг 5: Массовое создание контента

Продемонстрируйте создание нескольких ресурсов одновременно:

```
Create multi-channel content for AgentKits launch:
1. LinkedIn announcement post
2. Twitter thread (5 tweets)
3. Email subject lines (5 A/B variations)
4. Google Ads headlines (5 variations, max 30 chars)
```

### Шаг 6: Перекрестная проверка с руководством по стилю

Покажите, как использовать руководство по стилю контента:

```
Read docs/content-style-guide.md
```

Обратите внимание на:
- Формулы заголовков (фреймворк 4-U)
- Шаблоны призывов к действию
- Стандарты читаемости
- Рекомендации по SEO-написанию

### Шаг 7: Команды быстрого доступа

Поделитесь основными шаблонами команд:

**Команды для кампаний:**
- `/campaign:plan` - Создать план кампании
- `/campaign:brief` - Сгенерировать креативный бриф
- `/campaign:analyze` - Проанализировать эффективность
- `/campaign:calendar` - Контент-календарь

**Команды для контента:**
- `/content:blog` - SEO-статья в блог
- `/content:social` - Контент для конкретной социальной платформы
- `/content:email` - Текст для email
- `/content:landing` - Текст для лендинга
- `/content:ads` - Рекламный текст
