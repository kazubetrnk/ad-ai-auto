**Модуль 1.7: Навигация и поиск**

Эффективная навигация критически важна по мере роста ваших маркетинговых проектов. Давайте освоим поиск команд и управление рабочими процессами.

**Продолжительность:** ~20 минут

---

### Шаг 1: Поиск команд

Практикуйте обнаружение команд:

```
What commands are available for campaigns?
```

```
Show me all SEO-related commands
```

```
What commands help with email marketing?
```

### Шаг 2: Справочник по категориям команд

Просмотрите все категории команд:

| Категория | Команды | Сценарии использования |
|----------|----------|-----------|
| `/campaign:*` | plan, brief, analyze, calendar | Управление кампаниями |
| `/content:*` | blog, social, email, landing, ads | Создание контента |
| `/seo:*` | keywords, audit, optimize, competitor | SEO оптимизация |
| `/analytics:*` | roi, funnel, report | Отслеживание эффективности |
| `/sales:*` | pitch, outreach, battlecard, qualify | Поддержка продаж |
| `/leads:*` | score, nurture, qualify | Управление лидами |
| `/crm:*` | sequence, segment, score, lifecycle | CRM автоматизация |
| `/sequence:*` | welcome, nurture, re-engage | Email последовательности |
| `/social:*` | engage, viral, schedule | Социальные медиа |
| `/research:*` | market, persona, trend | Исследование рынка |
| `/brand:*` | voice, book, assets | Управление брендом |
| `/ops:*` | daily, weekly, monthly | Операции |
| `/report:*` | weekly, monthly | Отчетность |
| `/checklist:*` | campaign-launch, seo-weekly | Чек-листы |

### Шаг 4: Поиск документации

Практикуйте поиск соответствующей документации:

```

```

```
Where can I find campaign playbooks?
```

```

```

### Шаг 5: Команды быстрого доступа

Поделитесь основными сокращенными командами:

**Быстрый контент:**
```
/content:fast "quick headline for social post"
```

**Качественный контент:**
```
/content:good "detailed blog post with research"
```

**Мозговой штурм:**
```
/brainstorm "marketing channel ideas for B2B SaaS"
```

### Шаг 6: Операционные команды

Практикуйте операционные рабочие процессы:

**Ежедневные операции:**
```
/ops:daily "content creation"
```

**Еженедельный обзор:**
```
/ops:weekly "current week dates"
```

**Чек-листы кампаний:**
```
/checklist:campaign-launch "Q1 Product Launch" "next week"
```

### Шаг 7: Полный рабочий процесс команд

Продемонстрируйте полный рабочий процесс:

```
1. /research:market "B2B productivity software"
2. /campaign:plan "Q2 Growth Campaign"
3. /seo:keywords "team productivity"
4. /content:blog "productivity topic" "target keyword"
5. /seo:optimize "the blog post" "keyword"
6. /content:social "blog topic" "linkedin"
7. /sequence:nurture "product" "blog readers"
8. /analytics:report "Q2" "all channels"
```
