## Инструкции для Claude

Научите студентов использовать агентов-рецензентов для контроля качества и получения обратной связи с разных точек зрения.

### Обзор урока

---

**Модуль 1.5: Глубокое погружение в агентов-рецензентов**

Маркетинговый набор включает 6 специализированных агентов-рецензентов. Они предоставляют экспертную обратную связь с разных точек зрения для обеспечения качества контента.

**Продолжительность:** ~30 минут

---

### Шаг 1: Зачем нужны агенты-рецензенты?

Каждый элемент маркетинга должен быть проверен с нескольких сторон:
- Соответствие бренду
- SEO-оптимизация
- Потенциал конверсии
- Соответствие персоне
- Практическая реализация

Агенты-рецензенты предоставляют вам экспертов по требованию для каждой области.

### Шаг 2: Эксперты-рецензенты

Объясните трех экспертов-рецензентов:

**`brand-voice-guardian`:**
- Обеспечивает согласованность голоса бренда
- Проверяет тон и сообщения
- Проверяет соответствие руководствам по бренду
- Отмечает контент, не соответствующий бренду

**`seo-specialist`:**
- Оценивает использование ключевых слов
- Проверяет элементы внутренней SEO-оптимизации
- Анализирует структуру контента
- Выявляет возможности для оптимизации

**`conversion-optimizer`:**
- Оценивает эффективность призывов к действию
- Анализирует элементы убеждения
- Выявляет точки трения
- Предлагает улучшения для конверсии

### Шаг 3: Рецензенты-персоны

Объясните трех рецензентов-персон:

**`manager-maria` (Менеджер по маркетингу, 38 лет):**
- Перспектива B2B компании среднего размера
- Оценивает пригодность для внедрения командой
- Учитывает ограничения ресурсов
- Анализирует осуществимость кампании

**`solo-steve` (Индивидуальный предприниматель, 32 года):**
- Перспектива бизнеса одного человека
- Оценивает эффективность по времени
- Анализирует простоту самостоятельного использования
- Учитывает чувствительность к бюджету

**`startup-sam` (Основатель стартапа, 28 лет):**
- Перспектива стартапа на ранней стадии
- Оценивает потенциал роста
- Анализирует факторы виральности
- Оценивает скорость выхода на рынок

### Шаг 4: Используйте экспертов-рецензентов

Практика с экспертами-рецензентами:

```
Проверьте нашу статью в блоге об AgentKits с экспертами-рецензентами:

1. `brand-voice-guardian`: Оцените согласованность бренда
2. `seo-specialist`: Оцените SEO-оптимизацию
3. `conversion-optimizer`: Оцените потенциал конверсии

Каждый должен поставить оценку (1-10) и предоставить:
- Топ-3 сильные стороны
- Топ-3 необходимые улучшения
- Конкретные рекомендуемые правки
```

### Шаг 5: Используйте рецензентов-персон

Практика с рецензентами-персонами:

```
Проверьте нашу email-последовательность AgentKits с точки зрения персон:

1. `manager-maria`: Найдет ли это ценным B2B менеджер по маркетингу?
2. `solo-steve`: Внедрит ли это индивидуальный предприниматель?
3. `startup-sam`: Обеспечит ли это рост для стартапа?

Каждый должен оценить:
- Резонанс с их потребностями
- Осуществимость внедрения
- Восприятие ценности
```

### Шаг 6: Объединенная комиссия по рецензированию

Используйте всех рецензентов вместе для всесторонней обратной связи:

```
Проведите полную комиссию по рецензированию текста целевой страницы AgentKits:

Экспертные обзоры:
- Согласованность бренда (brand-voice-guardian)
- SEO-оптимизация (seo-specialist)
- Потенциал конверсии (conversion-optimizer)

Обзоры персон:
- Соответствие B2B (manager-maria)
- Потенциал самообслуживания (solo-steve)
- Потенциал роста (startup-sam)

Синтезируйте: Какие изменения удовлетворят всех рецензентов?
```

### Шаг 7: Итеративное улучшение

Покажите цикл рецензирования-улучшения:

```
1. Создайте контент
2. Получите обратную связь от нескольких рецензентов
3. Внедрите изменения с наивысшим приоритетом
4. Повторно проверьте измененный контент
5. Повторяйте, пока все рецензенты не поставят оценку 8+
```

### Шаг 8: Когда использовать какого рецензента

Предоставьте руководство:

| Тип контента | Основные рецензенты |
|-------------|-------------------|
| Статьи в блоге | seo-specialist, brand-voice-guardian |
| Целевые страницы | conversion-optimizer, manager-maria |
| Email-последовательности | email-wizard, startup-sam |
| Контент для соцсетей | brand-voice-guardian, solo-steve |
| Материалы для продаж | sales-enabler, manager-maria |

- Эксперты-рецензенты проверяют параметры качества

- Проводите контент через циклы рецензирования
