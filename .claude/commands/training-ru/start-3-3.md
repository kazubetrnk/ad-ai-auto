# Модуль 3, Урок 3: CRO всплывающих окон и онбординга

## Конверсия посетителей и активация пользователей

Этот урок охватывает две критические точки конверсии: захват посетителей с помощью всплывающих окон и активация новых регистраций через онбординг.

## Цели обучения

По завершении этого урока вы сможете:
1. Разрабатывать высококонверсионные всплывающие окна, не раздражая пользователей
2. Оптимизировать потоки онбординга после регистрации
3. Определять и ускорять момент "Ага"
4. Создавать экраны платного доступа и обновления

---

## CRO всплывающих окон

### Когда всплывающие окна работают

| Тип | Триггер | Лучше всего для |
|------|---------|----------|
| При выходе | Курсор покидает область просмотра | Захват лидов, сохранение покидающих |
| С задержкой | 30-60 секунд на странице | Вовлеченные посетители |
| При прокрутке | 50-70% глубины прокрутки | Вовлеченность в контент |
| При клике | Действие пользователя | Конкретные CTA |

### Когда всплывающие окна не работают

- Появление сразу при загрузке страницы
- Нет четкого ценностного предложения
- Сложно закрыть
- Одно и то же окно при каждом посещении

---

## Упражнение по дизайну всплывающих окон

Используйте `/cro:popup` для разработки эффективных всплывающих окон:

```bash
/cro:popup "Design exit-intent popup for AgentKits blog. Goal: capture emails for 'Remote Team Productivity Guide' lead magnet."
```

### Элементы хорошего всплывающего окна

1. **Четкая ценность:** Что они получат
2. **Минимум полей:** Только email
3. **Легкое закрытие:** Видимый X
4. **Адаптивность для мобильных:** CTA в зоне досягаемости большого пальца
5. **Ограничение частоты:** Один раз за сессию

## CRO онбординга

### Уравнение активации

**Момент "Ага"** = Первый раз, когда пользователь получает основную ценность

Для AgentKits: "Когда член команды видит расписание фокуса своей команды и блокирует время без отвлечений"

### Паттерны онбординга

| Паттерн | Лучше всего для |
|---------|----------|
| Мастер настройки | Сложные продукты, требующие конфигурации |
| Чек-лист | Приложения с богатым функционалом |
| Интерактивный тур | Продукты с насыщенным UI |
| Галерея шаблонов | Креативные инструменты |
| Пример проекта | Инструменты на основе проектов |

## CRO платного доступа и обновления

Для freemium и пробных продуктов экраны обновления критически важны.

### Триггеры платного доступа

| Триггер | Контекст |
|---------|---------|
| Ограничение функции | Пользователь пытается использовать премиум-функцию |
| Лимит использования | Достигнут лимит бесплатного тарифа |
| Истечение пробного периода | Истекает пробный период |
| Предложение обновления | После момента получения ценности |

### Упражнение по платному доступу

```bash
/cro:paywall "Design upgrade screen for AgentKits. Trigger: user hits 5-user limit on free tier. Goal: convert to Team plan ($12/user)."
```

## Полная воронка CRO

Теперь вы можете оптимизировать полную воронку конверсии:

```
Посетитель → CRO страницы → CRO формы → CRO регистрации
     ↓
  CRO всплывающих окон (захват покидающих)
     ↓
Новый пользователь → CRO онбординга → Активация
     ↓
Бесплатный пользователь → CRO платного доступа → Платный клиент
```

Каждый навык обрабатывает конкретный этап.
