**Модуль 2.1: Написание брифа кампании**

Добро пожаловать в Модуль 2! Теперь мы применяем все, что вы изучили, к реальным маркетинговым рабочим процессам. Брифы кампаний являются основой успешного выполнения.

**Продолжительность:** ~45 минут

---

### Шаг 1: Объясните совместный подход

> Claude - ваш стратегический партнер, а не замена вашей маркетинговой экспертизы. Вы привносите идеи, знание рынка и стратегическое мышление. Claude помогает сформулировать и структурировать эти идеи.

### Шаг 2: Соберите стратегическую информацию

Спросите у студента о его стратегическом мышлении:

```
Давайте создадим комплексный бриф кампании для кампании роста AgentKits во втором квартале.

Сначала расскажите мне о своем стратегическом видении:
- Какова основная цель? (например, 2000 регистраций на пробную версию)
- Каков бюджет?
- Каковы сроки?
- Есть ли конкретные каналы, на которых нужно сосредоточиться?
- Какое ключевое сообщение в этом квартале?
```

Дождитесь их ответа, затем продолжайте.

### Шаг 3: Используйте команду планирования кампании

Используйте команду планирования кампании:

```
/campaign:plan "AgentKits Q2 Growth - Goal: 2000 trial signups, Budget: $75K, Timeframe: 8 weeks, Channels: LinkedIn Ads, Content Marketing, Email Nurture, Key message: Team-wide focus time coordination"
```

Изучите комплексный план кампании, созданный агентом `planner`.

### Шаг 4: Создайте креативный бриф

Теперь используйте команду для креативного брифа:

```
/campaign:brief "AgentKits Q2 Growth Campaign"
```

Объясните, что включает креативный бриф:
- Единое ключевое предложение
- Инсайты целевой аудитории
- Тон и манера
- Список результатов
- Обязательные креативные элементы

### Шаг 5: Получите обратную связь с разных точек зрения

Используйте агентов-рецензентов:

```
Review the Q2 campaign plan from three perspectives:

1. `manager-maria` (Marketing Manager) - Is this executable by a marketing team?
2. `conversion-optimizer` - Will this campaign structure drive conversions?
3. `brand-voice-guardian` - Is the messaging on-brand?

Provide specific feedback and recommendations.
```

### Шаг 6: Создайте контент-календарь

Используйте команду календаря:

```
/campaign:calendar "8 weeks - AgentKits Q2 Growth - content marketing, social media, email nurture focused on team productivity"
```

### Шаг 7: Создайте вспомогательные документы

Создайте дополнительные активы кампании:

**Модель оценки лидов:**
```
/leads:score "B2B SaaS - technology companies - team productivity"
```

**Приветственная последовательность:**
```
/sequence:welcome "AgentKits" "trial signups from Q2 campaign"
```

**Последовательность развития:**
```
/sequence:nurture "AgentKits" "engaged leads who haven't converted"
```

### Шаг 8: Подготовка к конкурентной борьбе

Подготовьте конкурентные материалы:

```
/sales:battlecard "RescueTime - main competitor for productivity tools"
```

```
/competitor:deep "Freedom app - focus and productivity blocking"
```

### Шаг 9: Создайте план измерений

Настройте аналитику:

```
/analytics:funnel "trial signup funnel - awareness to trial to paid"
```

### Шаг 10: Сохраните как шаблон

Объясните, что этот рабочий процесс можно повторить для любой кампании:

```
Campaign Brief Workflow:
1. /campaign:plan - Strategic plan
2. /campaign:brief - Creative brief
3. /campaign:calendar - Content calendar
4. /leads:score - Lead qualification
5. /sequence:welcome - New lead nurture
6. /sequence:nurture - Ongoing nurture
7. /sales:battlecard - Competitive prep
8. /analytics:funnel - Measurement setup
```

## Ключевые моменты обучения
- Брифы кампаний создаются совместно
- Используйте `/campaign:plan` для стратегического планирования
- Используйте `/campaign:brief` для креативного направления
- Используйте `/campaign:calendar` для планирования контента
- Используйте агентов-рецензентов для обратной связи
- Создавайте вспомогательные активы (оценка лидов, последовательности, боевые карты)
