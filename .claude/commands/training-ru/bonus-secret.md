### Столп 1: Мастерство Команд

Знайте, какие команды связывать вместе:

**Рабочий Процесс Запуска Кампании:**
```
1. /research:market → Understand market
2. /research:persona → Define audience
3. /campaign:plan → Strategic plan
4. /campaign:brief → Creative direction
5. /campaign:calendar → Content schedule
6. /seo:keywords → SEO foundation
7. /content:blog → Pillar content
8. /content:social → Distribution content
9. /sequence:welcome → Lead nurture
10. /analytics:funnel → Measurement
```

**Рабочий Процесс Производства Контента:**
```
1. /seo:keywords → Find opportunities
2. /content:blog → Create content
3. /seo:optimize → Optimize for search
4. /content:social → Repurpose for social
5. Review with agents → Quality check
```

### Столп 2: Оркестрация Агентов

Используйте агентов стратегически:

**Для Фазы Исследования:**
- `researcher` → Понимание рынка
- `attraction-specialist` → Конкурентная разведка

**Для Фазы Создания:**
- `brainstormer` → Концепции кампаний
- `copywriter` → Создание контента
- `planner` → Стратегическая структура

**Для Фазы Проверки:**
- `brand-voice-guardian` → Проверка голоса бренда
- `conversion-optimizer` → CRO проверка
- `seo-specialist` → SEO валидация
- Рецензенты персон → Соответствие аудитории

**Для Фазы Оптимизации:**
- `lead-qualifier` → Качество лидов
- `email-wizard` → Оптимизация последовательностей
- `continuity-specialist` → Удержание

## Система Выполнения 10x

### Ежедневно: Быстрые Победы (15 мин)
```
/ops:daily "focus area"
```
- Проверить метрики
- Создать 1 единицу контента
- Обновить библиотеку паттернов

### Еженедельно: Стратегический Обзор (1 час)
```
/ops:weekly "current week"
```
- Обзор производительности кампании
- Планирование контента на следующую неделю
- Обновление последовательностей на основе данных

### Ежемесячно: Глубокий Анализ (2 часа)
```
/ops:monthly "current month"
/analytics:report "all campaigns" "full analysis"
```
- Комплексный обзор производительности
- Обновление документации
- Уточнение стратегий
