### Шаг 1: Создайте контекст

Объясните их роль:

> Вы - маркетинговый стратег в агентстве **Markit**. Ваш клиент - **AgentKits**. Ваша миссия:
> 1. Запустить продукт на рынок
> 2. Повысить осведомленность и количество регистраций
> 3. Создавать контент, который находит отклик у удаленных команд
> 4. Построить устойчивую систему контент-маркетинга

### Шаг 2: Изучите основные рабочие процессы

Объясните три основных рабочих процесса в `.claude/workflows/`:

**Маркетинговый конвейер (`primary-workflow.md`):**
```
Research → Insights → Creative → Plan → Create → Edit → Publish → Measure
```

**Воронка продаж (`sales-workflow.md`):**
```
Lead → MQL → SQL → Opportunity → Proposal → Negotiation → Close
```

**Жизненный цикл CRM (`crm-workflow.md`):**
```
Subscriber → Lead → MQL → SQL → Opportunity → Customer → Advocate
```

### Шаг 3: Изучите роли агентов

Объясните, как агенты соответствуют маркетинговым функциям:

**TOFU (Верх воронки):**
- `attraction-specialist` - Генерация лидов, SEO, целевые страницы

**MOFU (Середина воронки):**
- `lead-qualifier` - Определение намерений, оценка лидов
- `email-wizard` - Цепочки взращивания

**BOFU (Низ воронки):**
- `sales-enabler` - Презентации, кейс-стади, боевые карты

**Удержание:**
- `continuity-specialist` - Обнаружение оттока, повторное вовлечение
- `upsell-maximizer` - Расширение дохода

### Шаг 4: Создайте первый бриф кампании

Теперь начинается настоящая работа с использованием `/campaign:plan`:

```
/campaign:plan "AgentKits Q1 Product Launch - Target: 1000 trial signups in 30 days, Budget: $50K, Channels: LinkedIn, Google Ads, Content, Email"
```

Проверьте созданный комплексный план кампании.

### Шаг 5: Изучите бриф

Отметьте, как агент-планировщик:
- Создал структурированные цели и KPI
- Определил сегменты целевой аудитории
- Распределил бюджет по каналам
- Настроил систему измерений

### Шаг 6: Сила итерации

Покажите им улучшение с помощью дополнительных вопросов:

```
Expand the target audience section with day-in-the-life scenarios for each persona
```

Объясните: Итерация - это ключ. Первые черновики - это отправные точки.

### Шаг 7: Демонстрация понимания контекста

Продемонстрируйте силу контекста:

```
/content:social "Product launch announcement for AgentKits based on the campaign brief" "linkedin"
```

Покажите, как Claude автоматически использует информацию из контекста кампании.
