## Инструкции для Claude

Проведите студента через процесс проверки установки Claude Code и настройки маркетингового набора.

### Обзор урока

Скажите что-то вроде:

---

**Модуль 0.1: Установка и настройка**

Прежде чем мы углубимся в маркетинговые рабочие процессы, давайте убедимся, что всё настроено правильно.

---

### Шаг 1: Проверка Claude Code

Попросите их подтвердить:
- Они запускают это внутри Claude Code (а не в веб-чате)
- У них есть подписка Claude Pro или Max

Если они не уверены, объясните:
- Claude Code — это терминальная/CLI версия
- Она может читать, записывать и редактировать файлы напрямую
- Это отличается от веб-чата claude.ai

### Шаг 2: Проверка файлов маркетингового набора

Выполните эти проверки ВМЕСТЕ со студентом (фактически выполните их):

```
Show me the contents of this directory
```

Они должны увидеть:
- Папку `.claude/` с агентами, командами, навыками, рабочими процессами
- Папку `docs/` с документацией
- Файл `CLAUDE.md` (память проекта)
- Файл `README.md`

### Шаг 3: Изучение структуры системы

Покажите им структуру маркетингового набора:

```
List all folders in .claude/
```

Объясните каждый компонент:
- `agents/` - 18 специализированных маркетинговых агентов
- `commands/` - 76 слэш-команд, организованных по функциям
- `skills/` - Знания в области маркетинга
- `workflows/` - Основные рабочие процессы маркетинга, продаж и CRM

### Шаг 4: Изучение доступных команд

Покажите им категории команд:

```
List all folders in .claude/commands/
```

Объясните ключевые группы команд:
- `campaign/` - `/campaign:plan`, `/campaign:brief`, `/campaign:analyze`
- `content/` - `/content:blog`, `/content:social`, `/content:email`, `/content:landing`
- `seo/` - `/seo:keywords`, `/seo:audit`, `/seo:optimize`
- `analytics/` - `/analytics:roi`, `/analytics:funnel`, `/analytics:report`
- `sales/` - `/sales:pitch`, `/sales:outreach`, `/sales:battlecard`

### Шаг 5: Тестирование первой команды

Попросите их попробовать настоящую команду:

```
/brainstorm "What are the best marketing channels for a B2B SaaS product?"
```

Отпразднуйте выполнение их первой команды!

### Шаг 6: Просмотр документации

Покажите им ключевые документы:

```
Read docs/usage-guide.md (first 50 lines)
```

Объясните:
- `docs/usage-guide.md` - Полный справочник системы
- `docs/brand-guidelines.md` - Шаблон стандартов бренда
- `docs/content-style-guide.md` - Стандарты написания
- `docs/campaign-playbooks.md` - Шаблоны кампаний
- `docs/channel-strategies.md` - Тактики платформ
- `docs/analytics-setup.md` - Настройка отслеживания
