## Что такое библиотека паттернов?

Библиотека паттернов документирует:
- **Успешные паттерны** - Что сработало и почему
- **Неудачные паттерны** - Что не сработало и почему
- **Паттерны для конкретных персон** - Что находит отклик у каждой аудитории
- **Паттерны для конкретных каналов** - Что работает на каждой платформе
- **Сезонные паттерны** - Что работает в разное время

## Шаблон библиотеки паттернов

```markdown
# Marketing Pattern Library

## How to Use
- Review before each campaign
- Update after each campaign
- Track performance metrics

## Headline Patterns

### High-Performers (2x+ average)
| Pattern | Example | Performance | Context |
|---------|---------|-------------|---------|
| How [Persona] can [Benefit] | "How remote managers save 10 hrs/week" | 2.3x CTR | Works best for Solo Sam |
| [Number] ways to [Outcome] | "7 ways to coordinate team focus" | 2.1x CTR | Odd numbers outperform even |
| The [Adjective] guide to [Topic] | "The complete guide to remote productivity" | 1.8x CTR | Best for pillar content |

### Underperformers (Avoid)
| Pattern | Why It Failed |
|---------|---------------|
| "Revolutionary" anything | Too hyperbolic, low trust |
| Clickbait questions | High opens, low engagement |

| Pattern | Open Rate | Notes |
|---------|-----------|-------|

- Single CTA outperforms multiple 3:1

| Day | Time | Performance |
|-----|------|-------------|
| Tuesday | 10:00 AM | Best overall |
```

## Использование команд для создания паттернов

После каждой кампании используйте:

```
/analytics:report "campaign name" "all metrics"
```

Затем задокументируйте:
1. Какой контент показал лучшие результаты?
2. Какие заголовки получили самый высокий CTR?
3. Какие призывы к действию привели к наибольшему числу конверсий?
4. Какое время/дни работали лучше всего?

## Лучшие практики библиотеки паттернов

1. **Просматривайте перед кампаниями** - Проверьте, что работало раньше
2. **Обновляйте после кампаний** - Добавляйте новые знания
3. **Включайте неудачи** - Знайте, чего следует избегать
4. **Добавляйте контекст** - Важно понимать, почему это сработало
5. **Отслеживайте метрики** - Количественно измеряйте эффективность

## Паттерны CRO

### Паттерны посадочных страниц

| Элемент | Успешный паттерн | Улучшение |
|---------|-----------------|-------------|
| Заголовок | Конкретная выгода + число | +40% вовлеченности |
| Подзаголовок | "Как" или "Для кого это" | +25% ясности |
| Призыв к действию | От первого лица ("Начать мою пробную версию") | +28% кликов |
| Поля формы | Максимум 3-4 поля | +35% завершений |
| Социальное доказательство | Логотипы + "Присоединяйтесь к X командам" | +22% доверия |

### Паттерны форм

| Паттерн | Влияние |
|---------|--------|
| Одноколоночная компоновка | +15% завершений |
| Индикатор прогресса (многошаговый) | +20% завершений |
| Встроенная валидация | +22% сокращения ошибок |
| Умолчания по умолчанию | +10% завершений |
| "Кредитная карта не требуется" | +18% регистраций |

### Паттерны всплывающих окон

| Тип | Время срабатывания | Конверсия |
|------|--------|------------|
| При выходе | Мышь покидает область просмотра | 3-5% захвата |
| При прокрутке на 50% | Вовлеченные посетители | 2-4% захвата |
| Задержка 30 секунд | После чтения | 1-3% захвата |

**Элементы высококонверсионных всплывающих окон:**
- Четкая ценность в заголовке (не "Подписаться")
- Одно поле (только email)
- Видимая кнопка закрытия
- Адаптивность для мобильных (полноэкранный режим допустим)
- Ограничение частоты (один раз за сеанс)

### Паттерны процесса регистрации

| Паттерн | Влияние на конверсию |
|---------|-------------------|
| Начало только с email | +45% первоначального захвата |
| Опция Google/SSO | +30% для потребителей |
| Прогрессивное профилирование | +25% по сравнению с длинной формой |
| Опция показа пароля | +12% завершений |

### Паттерны онбординга

| Элемент | Влияние на активацию |
|---------|-------------------|
| Пустое состояние с ценностью | +35% уровня действий |
| Примеры данных/шаблоны | +28% исследования |
| Чек-лист из 3 шагов | +40% завершений |
| Aha-момент менее чем за 5 минут | +50% конверсии пробной версии |

### Паттерны платного доступа

| Триггер | Конверсия |
|---------|------------|
| После момента ценности | Самая высокая (2-4%) |
| Ограничение функции | Средняя (1-2%) |
| Лимит использования | Переменная |
| Истечение пробного периода | Зависит от вовлеченности |

**Эффективные элементы экрана обновления:**
- Таблица сравнения функций
- Выделен текущий план
- Отзыв от похожего пользователя
- Значок "Самый популярный"
- Годовая скидка на видном месте
