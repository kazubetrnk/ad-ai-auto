**Модуль 1.3: Первые маркетинговые задачи**

Теперь давайте выполним реальную маркетинговую работу. Вы выполните три распространенные задачи, которые каждый маркетолог выполняет регулярно.

**Продолжительность:** ~30 минут

---

### Задача 1: Создание текстов для нескольких каналов

Создайте тексты для нескольких каналов, используя команды для контента:

**Пост в LinkedIn:**
```
/content:social "AgentKits launch announcement - team productivity coordination for remote teams" "linkedin"
```

**Блог-пост:**
```
/content:blog "How Remote Teams Can Coordinate Focus Time Without Endless Meetings" "remote team coordination"
```

**Email:**
```
/content:email "product announcement" "existing subscribers"
```

Просмотрите результаты вместе. Покажите итерацию:

```
Make the LinkedIn post more educational - focus on the problem of coordinating deep work time
```

### Задача 2: Конкурентный анализ

Используйте команду конкурентного анализа:

```
/competitor:deep "RescueTime - personal time tracking and productivity software"
```

Объясните, что анализирует агент `researcher`:
- Целевая аудитория
- Ключевые функции и позиционирование
- Ценовая модель
- Сильные и слабые стороны
- Рыночные возможности

Задайте уточняющий вопрос:
```
Based on this analysis, what's AgentKits's biggest competitive advantage?
```

### Задача 3: Контент-календарь

Используйте команду календаря кампании:

```
/campaign:calendar "4 weeks - AgentKits product launch - focus on remote work productivity, team collaboration, deep work"
```

Просмотрите созданный календарь:
- Темы блог-постов с SEO-ключевыми словами
- Темы для социальных сетей по платформам
- Расписание email-рассылок
- Цели контента для каждого элемента

### Шаг 4: Расширьте один элемент

Возьмите тему и расширьте её, используя команды для контента:

```
/content:blog "The Complete Guide to Team Focus Time: How Remote Teams Can Coordinate Deep Work" "team focus time"
```

### Шаг 5: SEO-оптимизация

Используйте SEO-команды для оптимизации:

```
/seo:keywords "remote team productivity"
```

Затем:
```
/seo:optimize "the blog post we just created" "team focus time"
```

### Шаг 6: Проверка со специалистами

Используйте агентов-рецензентов (объясните, что они будут подробно рассмотрены позже):

```
Review the blog post from three perspectives:
1. Brand Voice Guardian - does it match our voice?
2. SEO Specialist - is it optimized for search?
3. Conversion Optimizer - will it drive action?
```

## Ключевые моменты обучения
- Реальные команды выполняют реальные маркетинговые задачи
- Команды `/content:*` создают контент для конкретных платформ
- `/competitor:deep` предоставляет конкурентную разведку
- `/campaign:calendar` создает контент-календари
- Команды `/seo:*` занимаются поисковой оптимизацией
- Всегда предоставляйте контекст (бренд, аудитория, цели)
