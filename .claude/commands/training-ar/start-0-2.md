### نظرة عامة على الدرس

---

**الوحدة 0.2: مهمتك التسويقية الأولى**

الآن لنقم ببعض الأعمال التسويقية الحقيقية. سنقوم بإنشاء إرشادات العلامة التجارية لـ AgentKits باستخدام أمر `/brand:voice`.

---

### الخطوة 1: شرح إرشادات العلامة التجارية

اشرح لماذا تعتبر إرشادات العلامة التجارية مهمة:
- تضمن الاتساق عبر جميع المحتويات
- تساعد Claude (والبشر) على الكتابة بالأسلوب الصحيح
- توثق الرسائل والمصطلحات الرئيسية
- تمنع المحتوى الذي لا يتوافق مع العلامة التجارية

### الخطوة 2: استخدام أمر صوت العلامة التجارية

أرشدهم لاستخدام أمر النظام الفعلي:

```
/brand:voice "AgentKits - B2B team productivity coordination tool for remote teams"
```

دع Claude ينشئ إرشادات شاملة للعلامة التجارية، ثم راجعها مع الطالب.

### الخطوة 3: إنشاء شخصيات العملاء

الآن استخدم أمر البحث للشخصيات:

```
/research:persona "Remote team managers at tech companies using AgentKits"
```

اشرح:
- وكيل `researcher` يتولى أبحاث السوق
- الشخصيات تساعد في استهداف المحتوى لجماهير محددة
- سنستخدم هذه الشخصيات خلال الدورة

### الخطوة 4: مراجعة ما تم إنشاؤه

راجع المخرجات معاً:

```
Show me what the brand:voice command created
```

أشر إلى:
- سمات الصوت وطيف النبرة
- إطار الرسائل
- الكلمات التي يجب استخدامها وتجنبها
- التكامل مع الوكلاء الآخرين

### الخطوة 5: قوة السياق

اشرح:
- هذه الإرشادات موجودة الآن في سياق المشروع
- في المهام المستقبلية، يمكن للوكلاء الرجوع إليها
- هذا هو "الوعي بالسياق" - إحدى القوى الخارقة لـ Claude
- سنستخدمها خلال الدورة

### تمرين سريع: اختبار إنشاء المحتوى

اطلب منهم تجربة استخدام سياق العلامة التجارية:

```
/content:social "Remote team productivity tips" "linkedin"
```

أظهر كيف يستخدم Claude سياق العلامة التجارية تلقائياً في إنشاء المحتوى.

### الخطوة 6: استكشاف الأوامر الرئيسية الأخرى

اعرض بإيجاز الأوامر الأخرى التي سيتقنونها:

**تخطيط الحملات:**
```
/campaign:plan "Q1 Product Launch"
```

**بحث تحسين محركات البحث:**
```
/seo:keywords "remote team productivity"
```

**تسلسلات البريد الإلكتروني:**
```
/sequence:welcome "AgentKits" "trial users"
```
