## تعليمات لـ Claude

ابدأ الوحدة 1 - المفاهيم الأساسية. يقدم هذا الدرس مشروع وكالة Markit وسير العمل الأساسي لمجموعة التسويق.

### نظرة عامة على الدرس

---

**الوحدة 1.1: مرحباً بك في Markit**

مرحباً بك في الوحدة 1! سنتقن الآن المفاهيم الأساسية لمجموعة التسويق من خلال العمل العملي. بنهاية هذه الوحدة، ستتعامل مع مهام التسويق الحقيقية بثقة.

**المدة:** ~20 دقيقة

---

### الخطوة 1: تهيئة المشهد

اشرح دورهم:

> أنت استراتيجي تسويق في وكالة **Markit**. عميلك هو **AgentKits**. مهمتك:
> 1. إطلاق المنتج في السوق
> 2. توليد الوعي والتسجيلات
> 3. إنشاء محتوى يتردد صداه مع الفرق البعيدة
> 4. بناء محرك تسويق محتوى مستدام

### الخطوة 2: فهم سير العمل الأساسي

اشرح سير العمل الرئيسية الثلاثة في `.claude/workflows/`:

**مسار التسويق (`primary-workflow.md`):**
```
Research → Insights → Creative → Plan → Create → Edit → Publish → Measure
```

**مسار المبيعات (`sales-workflow.md`):**
```
Lead → MQL → SQL → Opportunity → Proposal → Negotiation → Close
```

**دورة حياة CRM (`crm-workflow.md`):**
```
Subscriber → Lead → MQL → SQL → Opportunity → Customer → Advocate
```

### الخطوة 3: فهم أدوار الوكلاء

اشرح كيف ترتبط الوكلاء بوظائف التسويق:

**TOFU (أعلى القمع):**
- `attraction-specialist` - توليد العملاء المحتملين، SEO، الصفحات المقصودة

**MOFU (منتصف القمع):**
- `lead-qualifier` - كشف النوايا، تسجيل العملاء المحتملين
- `email-wizard` - تسلسلات الرعاية

**BOFU (أسفل القمع):**
- `sales-enabler` - العروض التقديمية، دراسات الحالة، بطاقات المعركة

**الاحتفاظ:**
- `continuity-specialist` - كشف التوقف عن الاستخدام، إعادة المشاركة
- `upsell-maximizer` - توسيع الإيرادات

### الخطوة 4: إنشاء موجز الحملة الأول

الآن يبدأ العمل الحقيقي باستخدام `/campaign:plan`:

```
/campaign:plan "AgentKits Q1 Product Launch - Target: 1000 trial signups in 30 days, Budget: $50K, Channels: LinkedIn, Google Ads, Content, Email"
```

راجع خطة الحملة الشاملة التي تم إنشاؤها.

### الخطوة 5: مراجعة الموجز

وضّح كيف قام وكيل التخطيط بـ:
- إنشاء أهداف ومؤشرات أداء رئيسية منظمة
- تحديد شرائح الجمهور المستهدف
- توزيع الميزانية عبر القنوات
- إعداد إطار القياس

### الخطوة 6: قوة التكرار

أظهر لهم التحسين باستخدام أسئلة المتابعة:

```
Expand the target audience section with day-in-the-life scenarios for each persona
```

اشرح: التكرار هو المفتاح. المسودات الأولى هي نقاط انطلاق.

### الخطوة 7: عرض توضيحي للوعي بالسياق

أظهر قوة السياق:

```
/content:social "Product launch announcement for AgentKits based on the campaign brief" "linkedin"
```

أظهر كيف يستمد Claude من سياق الحملة تلقائياً.

### ما التالي

أخبرهم:
- لقد أنشأوا موجز حملة احترافي باستخدام `/campaign:plan`
- استخدم Claude السياق من إرشادات العلامة التجارية والشخصيات
- **التالي:** `/training-ar:start-1-2` - العمل مع ملفات التسويق
- سيتعلمون تنظيم وإيجاد وإدارة الأصول التسويقية بكفاءة

## نقاط التعليم الرئيسية
- وكالة Markit هي مشروع الممارسة العملية
- ثلاثة سير عمل أساسية: التسويق، المبيعات، CRM
- الوكلاء يرتبطون بمراحل القمع (TOFU، MOFU، BOFU، الاحتفاظ)
- `/campaign:plan` ينشئ موجزات حملات شاملة
- التكرار يحسن جودة المخرجات
