### نظرة عامة على الدرس

---

**الوحدة 1.4: استخدام الوكلاء للتسويق**

تحتوي مجموعة التسويق على 18 وكيلاً متخصصاً. فكر فيهم كأعضاء فريق ذكاء اصطناعي يمكنهم العمل على مهام تسويقية محددة بخبرة.

**المدة:** ~35 دقيقة

---

### الخطوة 1: شرح نظام الوكلاء

تحتوي مجموعة التسويق على وكلاء منظمين حسب الوظيفة:

**الوكلاء التسويقيون الأساسيون (6):**
| الوكيل | التركيز | حالات الاستخدام |
|-------|-------|-----------|
| `attraction-specialist` | قمة القمع، توليد العملاء المحتملين | تحسين محركات البحث، الصفحات المقصودة، معلومات المنافسين |
| `lead-qualifier` | اكتشاف النية | تسجيل العملاء المحتملين، التحليل السلوكي |
| `email-wizard` | التسويق عبر البريد الإلكتروني | التسلسلات، الأتمتة، التحسين |
| `sales-enabler` | دعم المبيعات | العروض التقديمية، دراسات الحالة، بطاقات المعركة |
| `continuity-specialist` | الاحتفاظ بالعملاء | كشف الانسحاب، إعادة المشاركة |
| `upsell-maximizer` | توسيع الإيرادات | البيع المتقاطع، البيع الإضافي، اعتماد الميزات |

**الوكلاء الداعمون (6):**
| الوكيل | التركيز | حالات الاستخدام |
|-------|-------|-----------|
| `researcher` | ذكاء السوق | البحث، التحليل التنافسي |
| `brainstormer` | الإبداع الفكري | مفاهيم الحملات، زوايا الرسائل |
| `planner` | التخطيط الاستراتيجي | خطط الحملات، تقاويم المحتوى |
| `project-manager` | التنسيق | تتبع الحالة، إشراف الحملة |
| `copywriter` | إنشاء المحتوى | النسخ، الرسائل، الإبداع |
| `docs-manager` | التوثيق | إرشادات العلامة التجارية، أدلة الأسلوب |

**وكلاء المراجعة (6):**
| الوكيل | المنظور | يراجع لـ |
|-------|-------------|-------------|
| `brand-voice-guardian` | اتساق العلامة التجارية | الصوت، النبرة، الرسائل |
| `conversion-optimizer` | خبير تحسين معدل التحويل | التحويل، الإقناع |
| `seo-specialist` | تحسين البحث | الكلمات المفتاحية، تحسين محركات البحث التقني |
| `manager-maria` | مدير تسويق (38 سنة، B2B) | الاستراتيجية، ملاءمة الفريق |
| `solo-steve` | رائد أعمال منفرد (32 سنة) | الوقت، الميزانية، الأعمال اليدوية |
| `startup-sam` | مؤسس شركة ناشئة (28 سنة) | النمو، الانتشار، السرعة |

### الخطوة 2: تمرين الوكلاء - المراجعة متعددة المنظورات

أنشئ محتوى وراجعه من منظورات متعددة:

```
راجع موجز حملة AgentKits من ثلاث منظورات للوكلاء:

1. بصفتك `brand-voice-guardian` - قيّم اتساق العلامة التجارية والرسائل
2. بصفتك `conversion-optimizer` - قيّم الدعوات للعمل، الإقناع، وإمكانية التحويل
3. بصفتك `manager-maria` - هل سيعمل هذا لفريق تسويق B2B لتنفيذه؟

لكل منظور، قدم:
- ما الذي يعمل بشكل جيد
- مجالات التحسين
- توصيات محددة
```

اشرح ما حدث للتو - ثلاث مراجعات متخصصة في أمر واحد.

### الخطوة 3: استخدام تأهيل العملاء المحتملين

أظهر وكيل `lead-qualifier` عبر الأوامر:

```
/leads:score "B2B SaaS company - technology industry"
```

```
/leads:qualify "AgentKits productivity tool"
```

أظهر كيف ينشئ `lead-qualifier`:
- معايير التسجيل الديموغرافي
- إشارات التسجيل السلوكي
- عتبات MQL/SQL

### الخطوة 4: استخدام معالج البريد الإلكتروني

أظهر وكيل `email-wizard`:

```
/sequence:welcome "AgentKits" "trial users"
```

```
/sequence:nurture "AgentKits" "leads who downloaded our guide"
```

### الخطوة 5: استخدام مُمكّن المبيعات

أظهر وكيل `sales-enabler`:

```
/sales:pitch "enterprise company considering AgentKits" "team coordination"
```

```
/sales:battlecard "RescueTime"
```

### الخطوة 6: سيناريو واقعي - الاستجابة السريعة

```
السيناريو: أعلن أحد المنافسين للتو عن ميزة "تركيز الفريق". استخدم الوكلاء للاستجابة:

1. استخدم `researcher` لتحليل إعلانهم
2. استخدم `brainstormer` لتطوير موقف مضاد
3. استخدم `copywriter` لإنشاء محتوى الاستجابة
4. استخدم `email-wizard` لصياغة اتصال العملاء
```

### الخطوة 7: أفضل ممارسات الوكلاء

شارك هذه النصائح:
- كن محدداً مع أهداف المهام
- اذكر إرشادات العلامة التجارية والشخصيات
- حدد المخرجات بوضوح (التنسيق، الطول)
- استخدم الوكلاء المتخصصين للمهام المتخصصة
- اجمع الوكلاء للمشاريع المعقدة
