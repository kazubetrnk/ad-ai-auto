## معايير اللغة والجودة

**مهم جداً**: استجب بنفس اللغة التي يستخدمها المستخدم. إذا كانت فيتنامية، استجب بالفيتنامية. إذا كانت إسبانية، استجب بالإسبانية.

---

## تعليمات لـ Claude

الدرس الأخير من الوحدة 1 - علّم التنقل الفعّال وإتقان الأوامر.

### نظرة عامة على الدرس

---

**الوحدة 1.7: التنقل والبحث**

التنقل الفعّال أمر بالغ الأهمية مع نمو مشاريعك التسويقية. لنتقن إيجاد الأوامر وإدارة سير العمل.

**المدة:** ~20 دقيقة

---

### الخطوة 1: إيجاد الأوامر

تدرّب على اكتشاف الأوامر:

```
What commands are available for campaigns?
```

```
Show me all SEO-related commands
```

```
What commands help with email marketing?
```

### الخطوة 2: مرجع فئات الأوامر

راجع جميع فئات الأوامر:

| الفئة | الأوامر | حالات الاستخدام |
|----------|----------|-----------|
| `/campaign:*` | plan, brief, analyze, calendar | إدارة الحملات |
| `/content:*` | blog, social, email, landing, ads | إنشاء المحتوى |
| `/seo:*` | keywords, audit, optimize, competitor | تحسين محركات البحث |
| `/analytics:*` | roi, funnel, report | تتبع الأداء |
| `/sales:*` | pitch, outreach, battlecard, qualify | تمكين المبيعات |
| `/leads:*` | score, nurture, qualify | إدارة العملاء المحتملين |
| `/crm:*` | sequence, segment, score, lifecycle | أتمتة CRM |
| `/sequence:*` | welcome, nurture, re-engage | تسلسلات البريد الإلكتروني |
| `/social:*` | engage, viral, schedule | وسائل التواصل الاجتماعي |
| `/research:*` | market, persona, trend | أبحاث السوق |
| `/brand:*` | voice, book, assets | إدارة العلامة التجارية |
| `/ops:*` | daily, weekly, monthly | العمليات |
| `/report:*` | weekly, monthly | إعداد التقارير |
| `/checklist:*` | campaign-launch, seo-weekly | قوائم التحقق |

### الخطوة 5: أوامر المرجع السريع

شارك اختصارات الأوامر الأساسية:

**محتوى سريع:**
```
/content:fast "quick headline for social post"
```

**محتوى جيد:**
```
/content:good "detailed blog post with research"
```

**العصف الذهني:**
```
/brainstorm "marketing channel ideas for B2B SaaS"
```

### الخطوة 6: الأوامر التشغيلية

تدرّب على سير العمل التشغيلي:

**العمليات اليومية:**
```
/ops:daily "content creation"
```

**المراجعة الأسبوعية:**
```
/ops:weekly "current week dates"
```

**قوائم التحقق من الحملات:**
```
/checklist:campaign-launch "Q1 Product Launch" "next week"
```

### الخطوة 7: سير العمل الكامل للأوامر

اعرض سير عمل كامل:

```
1. /research:market "B2B productivity software"
2. /campaign:plan "Q2 Growth Campaign"
3. /seo:keywords "team productivity"
4. /content:blog "productivity topic" "target keyword"
5. /seo:optimize "the blog post" "keyword"
6. /content:social "blog topic" "linkedin"
7. /sequence:nurture "product" "blog readers"
8. /analytics:report "Q2" "all channels"
```
