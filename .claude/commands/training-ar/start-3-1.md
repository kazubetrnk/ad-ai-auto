# الوحدة 3، الدرس 1: أساسيات CRO

## مرحباً بك في تحسين معدل التحويل

في هذه الوحدة، ستتقن مهارات CRO (تحسين معدل التحويل) الجديدة المضافة إلى AgentKits Marketing. تساعدك هذه المهارات على تحسين معدلات التحويل بشكل منهجي عبر جميع الأصول التسويقية.

## الأهداف التعليمية

بنهاية هذا الدرس، سوف:
1. تفهم فئات مهارات CRO الستة
2. تعرف متى تستخدم كل أمر CRO
3. تطبق المبادئ النفسية على التحويلات
4. تنشئ أول تدقيق CRO خاص بك

---

## مجموعة مهارات CRO

يتضمن AgentKits Marketing 7 مهارات CRO متخصصة:

| المهارة | استخدمها لـ | الأمر |
|-------|---------|---------|
| `page-cro` | الصفحات المقصودة، الصفحات الرئيسية، التسعير | `/cro:page` |
| `form-cro` | التقاط العملاء المحتملين، نماذج الاتصال والعروض التوضيحية | `/cro:form` |
| `popup-cro` | النوافذ المنبثقة، التراكبات، نية الخروج | `/cro:popup` |
| `signup-flow-cro` | التسجيل، الاشتراكات التجريبية | `/cro:signup` |
| `onboarding-cro` | التفعيل بعد التسجيل | `/cro:onboarding` |
| `paywall-upgrade-cro` | جدران الدفع داخل التطبيق، الترقيات | `/cro:paywall` |
| `ab-test-setup` | تصميم التجارب | `/test:ab-setup` |

---

## إطار عمل CRO

يتبع كل تحليل CRO هذا التسلسل الهرمي:

### 1. عرض القيمة (أعلى تأثير)
- هل يمكن للزوار فهم ما تقدمه في 5 ثوانٍ؟
- هل الفائدة واضحة، وليس فقط الميزات؟

### 2. فعالية العنوان الرئيسي
- هل ينقل القيمة الأساسية؟
- هل هو محدد وموثوق؟

### 3. تحسين CTA
- إجراء أساسي واحد واضح؟
- فوق الطية، مرئي، مقنع؟

### 4. إشارات الثقة
- دليل اجتماعي بالقرب من القرارات؟
- شارات الأمان مرئية؟

### 5. تقليل الاحتكاك
- الحد الأدنى من حقول النموذج؟
- خطوات تالية واضحة؟

## التمرين 1: تدقيق الصفحة المقصودة لـ AgentKits

لنطبق مبادئ CRO على الصفحة المقصودة لـ AgentKits.

### الحالة الحالية (افتراضية)

**العنوان الرئيسي:** "Team Productivity Software"
**CTA:** "Learn More"
**النموذج:** 7 حقول

### مهمتك

أنشئ تدقيق CRO باستخدام أمر `/cro:page`:

```bash
/cro:page "Analyze AgentKits's landing page: Headline 'Team Productivity Software', CTA 'Learn More', 7-field form. Target: remote team managers."
```

### الناتج المتوقع

يجب أن يحدد التدقيق:
- عنوان رئيسي عام (غير محدد أو موجه نحو الفوائد)
- CTA ضعيف ("Learn More" مقابل موجه نحو العمل)
- احتكاك عالٍ (7 حقول كثيرة جداً)

---

## التمرين 2: تطبيق علم النفس

تتضمن مهارة `marketing-psychology` أكثر من 70 نموذجاً ذهنياً. جرب:

```bash
/marketing:psychology "How can we use loss aversion and social proof to improve AgentKits's trial signup rate?"
```

### النماذج الرئيسية لـ CRO

| النموذج | التطبيق |
|-------|-------------|
| النفور من الخسارة | صياغة "لا تفوت الفرصة" |
| الدليل الاجتماعي | "انضم إلى أكثر من 10,000 فريق" |
| التثبيت | اعرض الخطة الأغلى أولاً |
| الندرة | أماكن أو وقت محدود |
| المعاملة بالمثل | قيمة مجانية قبل الطلب |

## مهمة التطبيق

أنشئ خطة تحسين CRO كاملة:

1. **قم بإجراء تدقيق الصفحة:**
   ```bash
   /cro:page "AgentKits homepage audit"
   ```

2. **قم بتحسين النموذج:**
   ```bash
   /cro:form "Reduce AgentKits's 7-field trial signup form"
   ```

3. **صمم اختبار A/B:**
   ```bash
   /test:ab-setup "Test new headline vs current for AgentKits"
   ```

احفظ عملك في:
```
training/exercises/markit/cro/
├── landing-page-audit.md
├── form-optimization.md
└── ab-test-plan.md
```
