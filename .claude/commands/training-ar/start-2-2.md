# /training-ar:start-2-2 - تطوير استراتيجية المحتوى

## معايير اللغة والجودة

**مهم جداً**: الرد بنفس اللغة التي يستخدمها المستخدم. إذا كانت فيتنامية، الرد بالفيتنامية. إذا كانت إسبانية، الرد بالإسبانية.

---

## تعليمات لـ Claude

علّم تطوير استراتيجية المحتوى الشاملة: البحث، التخطيط، الجداول الزمنية، والقياس.

### نظرة عامة على الدرس

---

**الوحدة 2.2: تطوير استراتيجية المحتوى**

استراتيجية المحتوى تحوّل إنشاء المحتوى العشوائي إلى محرك نمو منهجي. لنبني واحدة لـ AgentKits.

**المدة:** ~40 دقيقة

---

### الخطوة 1: أساس البحث

ابدأ ببحث السوق والجمهور:

```
/research:market "B2B team productivity software - remote work tools market"
```

```
/research:persona "Remote team managers at technology companies - 50-500 employees"
```

```
/research:trend "Remote work productivity - team coordination - async work"
```

### الخطوة 2: بحث الكلمات المفتاحية لتحسين محركات البحث

استخدم أوامر SEO لبناء أساس الكلمات المفتاحية:

```
/seo:keywords "remote team productivity"
```

```
/seo:keywords "team coordination software"
```

```
/seo:keywords "deep work for teams"
```

قم بتجميع الكلمات المفتاحية في مجموعات موضوعية:
- المجموعة 1: إنتاجية الفريق عن بُعد
- المجموعة 2: وقت التركيز للفريق
- المجموعة 3: التنسيق بدون اجتماعات

### الخطوة 3: تحليل محتوى المنافسين

قم بتحليل محتوى المنافسين:

```
/seo:competitor "rescuetime.com"
```

حدد:
- ما هي المواضيع التي يغطونها
- فجوات المحتوى التي يمكننا ملؤها
- الكلمات المفتاحية التي يتصدرون بها

### الخطوة 4: إنشاء تقويم المحتوى

استخدم تقويم الحملات لتخطيط المحتوى:

```
/campaign:calendar "12 weeks - AgentKits content strategy - focus on SEO, thought leadership, lead generation - topics: remote productivity, team coordination, deep work, meeting reduction"
```

### الخطوة 5: تحديد أنواع المحتوى

خطط للمحتوى حسب مراحل القمع:

**TOFU (الوعي):**
- مقالات المدونة (بتركيز على SEO)
- محتوى وسائل التواصل الاجتماعي
- قيادة الفكر

**MOFU (الاعتبار):**
- أدلة المقارنة
- محتوى إرشادي
- دراسات الحالة

**BOFU (القرار):**
- عروض المنتج التوضيحية
- حاسبات عائد الاستثمار
- شهادات العملاء

### الخطوة 6: إنشاء استراتيجية المحتوى الرئيسي

خطط لاستراتيجية الصفحة الرئيسية:

```
/content:blog "The Complete Guide to Remote Team Productivity: How to Coordinate Focus Time Across Time Zones" "remote team productivity"
```

محتوى المجموعات (اربط بالصفحة الرئيسية):
1. كيفية جدولة وقت التركيز للفريق
2. تقليل الاجتماعات دون فقدان التوافق
3. أفضل ممارسات التواصل غير المتزامن
4. العمل العميق في الفرق عن بُعد
5. تتبع الإنتاجية للفرق

### الخطوة 7: سير عمل إنتاج المحتوى

استخدم أوامر المحتوى لكل قطعة:

**إنتاج مقالات المدونة:**
```
1. /seo:keywords "topic" - بحث الكلمات المفتاحية
2. /content:blog "title" "keyword" - إنشاء المقال
3. /seo:optimize "post" "keyword" - التحسين
4. المراجعة مع وكيل seo-specialist
5. المراجعة مع وكيل brand-voice-guardian
```

**إنتاج محتوى وسائل التواصل:**
```
1. /content:social "topic" "linkedin" - منشور LinkedIn
2. /content:social "topic" "twitter" - سلسلة Twitter
3. المراجعة مع وكيل conversion-optimizer
```

### الخطوة 8: دمج البريد الإلكتروني

أنشئ تسلسلات البريد الإلكتروني لرعاية مستهلكي المحتوى:

```
/sequence:nurture "AgentKits" "blog readers who downloaded guide"
```

### الخطوة 9: خطة توزيع المحتوى

استخدم أوامر وسائل التواصل الاجتماعي للتوزيع:

```
/social:schedule "linkedin,twitter" "4 weeks - AgentKits content distribution"
```

### الخطوة 10: إطار القياس

قم بإعداد تحليلات المحتوى:

```
/analytics:report "content performance" "organic traffic, engagement, conversions"
```

المقاييس الرئيسية للتتبع:
- الزيارات العضوية حسب قطعة المحتوى
- الوقت المستغرق في الصفحة
- معدل التحويل لكل محتوى
- جودة العملاء المحتملين من المحتوى

### ما التالي

أخبرهم:
- لديهم الآن استراتيجية محتوى كاملة
- من النشر العشوائي إلى النمو المنهجي
- **التالي:** `/training-ar:start-2-3` - توليد النسخ التسويقية
- توسيع نطاق إنتاج النسخ مع الحفاظ على الجودة

## نقاط التدريس الرئيسية
- الاستراتيجية تحول المحتوى من عشوائي إلى منهجي
- أوامر `/research:*` تبني الأساس
- `/seo:keywords` يحدد الفرص
- الرئيسي + المجموعات = قوة SEO
- إنتاج المحتوى يتبع سير عمل قابل للتكرار
- القياس يضمن المساءلة
