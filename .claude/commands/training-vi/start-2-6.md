**Module 2.6: SEO Optimization**

SEO thúc đẩy tăng trưởng bền vững, dài hạn. Hãy thành thạo keyword research, content optimization, và technical SEO.

**Thời lượng:** ~40 phút

---

### Bước 1: Keyword Research

Sử dụng keyword research command:

```
/seo:keywords "năng suất team từ xa"
```

```
/seo:keywords "phần mềm thời gian tập trung team"
```

```
/seo:keywords "công cụ phối hợp cho remote teams"
```

Review keyword opportunities:
- Ước tính search volume
- Mức độ cạnh tranh
- Intent classification
- Keyword clusters

### Bước 2: Competitor SEO Analysis

Phân tích competitor SEO:

```
/seo:competitor "rescuetime.com"
```

```
/seo:competitor "freedom.to"
```

Xác định:
- Keywords họ rank
- Content gaps
- Backlink opportunities
- Technical strengths/weaknesses

### Bước 3: SEO Audit

Chạy comprehensive audit:

```
/seo:audit "agentkits.com"
```

Review audit areas:
- Technical SEO (crawlability, speed, mobile)
- On-page SEO (titles, meta, headers)
- Content quality (depth, relevance)
- Backlink profile

### Bước 4: Content Optimization

Tối ưu existing content:

```
/seo:optimize "blog post về team productivity" "năng suất team từ xa"
```

Optimization bao gồm:
- Cải thiện title tag
- Meta description
- Header structure (H1, H2, H3)
- Keyword placement
- Internal linking suggestions
- Image alt text

### Bước 5: Tạo SEO Content

Sử dụng blog command với SEO focus:

```
/content:blog "Hướng Dẫn Hoàn Chỉnh Về Năng Suất Team Từ Xa" "năng suất team từ xa"
```

Sau đó tối ưu:

```
/seo:optimize "blog post" "năng suất team từ xa"
```

### Bước 6: Featured Snippet Strategy

Tối ưu cho featured snippets:

```
Tạo content tối ưu cho featured snippets:

Target query: "Cách cải thiện năng suất remote team"

PARAGRAPH SNIPPET (40-60 từ):
Định dạng trả lời trực tiếp

LIST SNIPPET (numbered steps):
Định dạng step-by-step

TABLE SNIPPET:
Định dạng comparison
```

### Bước 7: Technical SEO Checklist

Sử dụng SEO checklist:

```
/checklist:seo-weekly "agentkits.com"
```

Weekly SEO tasks:
- Crawl error check
- Page speed monitoring
- Mobile usability
- Index status
- Keyword tracking

### Bước 8: Pillar-Cluster Strategy

Tạo pillar content plan:

```
Tạo pillar-cluster SEO strategy:

PILLAR PAGE: "Hướng Dẫn Hoàn Chỉnh Về Năng Suất Team Từ Xa"
- 3000-5000 từ, toàn diện

CLUSTER CONTENT (10 posts):
1. "Cách Lên Lịch Thời Gian Tập Trung Team" → team focus time
2. "Giảm Họp Mà Không Mất Alignment" → reduce meetings remote
3. "Async Communication Best Practices" → async communication
4. "Deep Work Cho Remote Teams" → deep work teams
5. "Team Productivity Metrics Quan Trọng" → productivity metrics
...

INTERNAL LINKING:
- Mỗi cluster links đến pillar
- Pillar links đến tất cả clusters
- Cross-links giữa related clusters
```

### Bước 9: Content Calendar với SEO

Tạo SEO-focused calendar:

```
/campaign:calendar "12 tuần - SEO content strategy - target keywords: năng suất team từ xa, thời gian tập trung team, phần mềm phối hợp"
```

Map content đến keywords:
- Tuần 1: [Keyword] - [Title] - [Type]
- Tuần 2: [Keyword] - [Title] - [Type]
...

### Bước 10: SEO Measurement

Track SEO performance:

```
/analytics:report "SEO performance" "organic traffic, keyword rankings, conversions"
```

Key metrics:
- Organic traffic trend
- Keyword position changes
- Organic conversion rate
- Page performance
