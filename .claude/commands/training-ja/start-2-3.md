# /training:start-2-3 - マーケティングコピーを生成する

## 言語と品質基準

**重要**: ユーザーが使用している言語で応答してください。日本語なら日本語で、スペイン語ならスペイン語で応答してください。

---

## Claudeへの指示

品質を維持しながらチャネル間で大量のコピー生成を教えてください。

### レッスン概要

---

**モジュール2.3: マーケティングコピーを生成する**

プロフェッショナルなマーケティングコピーを大規模に作成する方法を学びましょう：メール、広告、ソーシャル、ランディングページ。品質 + スピード。

**所要時間:** 約35分

---

### ステップ1: メールウェルカムシーケンス

シーケンスコマンドを使用してください：

```
/sequence:welcome "AgentKits" "トライアルユーザー - リモートチームマネージャー"
```

生成されたシーケンスをレビューしてください：
- メール1（0日目）：ウェルカム + クイックスタート
- メール2（2日目）：機能ハイライト
- メール3（5日目）：ソーシャルプルーフ + ヒント
- メール4（9日目）：価値の再確認
- メール5（13日目）：トライアル終了 + アップグレード

各メールに含まれるもの：
- A/Bテスト用の件名バリエーション
- プレビューテキスト
- 本文コピー
- 明確なCTA

### ステップ2: ソーシャルメディアコンテンツ

ソーシャル用のコンテンツコマンドを使用してください：

**LinkedIn：**
```
/content:social "リモートマネージャーのためのチーム調整のヒント - AgentKitsローンチ" "linkedin"
```

**Twitter：**
```
/content:social "リモートチームが調整に時間を浪費する5つの方法 - スレッド" "twitter"
```

### ステップ3: ブログコンテンツ

SEO重視でブログコマンドを使用してください：

```
/content:blog "リモートチームが終わりのない会議なしにフォーカスタイムを調整する方法" "リモートチームフォーカスタイム"
```

次に最適化：
```
/seo:optimize "ブログ投稿" "リモートチームフォーカスタイム"
```

### ステップ4: 有料広告コピー

広告コピーコマンドを使用してください：

**Google Ads：**
```
/content:ads "google" "チーム生産性ソフトウェア - サインアップを促進"
```

**Meta Ads：**
```
/content:ads "meta" "リモートチーム調整 - 認知キャンペーン"
```

**LinkedIn Ads：**
```
/content:ads "linkedin" "B2B生産性ツール - リード生成"
```

### ステップ5: ランディングページコピー

ランディングページコマンドを使用してください：

```
/content:landing "AgentKitsの14日間無料トライアル" "テクノロジー企業のリモートチームマネージャー"
```

これが生成するもの：
- ヒーローセクション（見出し、サブヘッド、CTA）
- 問題セクション
- ソリューションセクション
- メリット付き機能
- ソーシャルプルーフセクション
- 価格概要
- FAQセクション
- 最終CTA

### ステップ6: ファストvsグッドコンテンツ

2つのコンテンツモードを説明してください：

**ファストコンテンツ（`/content:fast`）：**
- 迅速な納品
- アイデア出しに最適
- 最初のドラフト
- 大量のニーズ

```
/content:fast "チームフォーカスタイムのメリットについてのクイックLinkedIn投稿"
```

**グッドコンテンツ（`/content:good`）：**
- 徹底的なリサーチ
- 複数のバリエーション
- 公開準備完了
- 戦略的なピース

```
/content:good "研究引用付きのチームフォーカスタイムの科学についての詳細なブログ投稿"
```

### ステップ7: コンテンツ強化

強化コマンドを使用してください：

```
/content:enhance "コピーをより会話的にして緊急性を追加"
```

```
/content:cro "より高いコンバージョン率に最適化"
```

### ステップ8: A/Bテストバリエーション

テストバリエーションを作成してください：

```
ランディングページのA/Bテストバリエーションを作成：

見出し（5つの角度）：
1. 結果重視
2. 問題重視
3. 質問
4. ハウツー
5. ソーシャルプルーフ

CTA（3バリエーション）：
1. 無料トライアルを開始
2. 無料で試す
3. 今すぐ始める
```

### ステップ9: ペルソナ別パーソナライゼーション

ペルソナ別バリエーションを作成してください：

**Solo Sam向け：**
```
/content:email "製品発表" "技術系チームマネージャー - 効率重視"
```

**Startup Sam向け：**
```
/content:email "製品発表" "スタートアップ創業者 - 成長とスケール重視"
```

### ステップ10: 品質レビュー

スペシャリストと全コンテンツをレビューしてください：

## 主な教育ポイント
- `/content:*`コマンドがすべてのコンテンツタイプを処理
- `/sequence:*`がメール自動化を作成
- ドラフトにはファストモード、最終版にはグッドモードを使用
- `/content:cro`がコンバージョンに最適化
- より高い関連性のためにペルソナ別にパーソナライズ
- 常にスペシャリストエージェントでレビュー
