# /training:start-1-2 - マーケティングファイルの操作

## 言語と品質基準

**重要**: ユーザーが使用している言語で応答してください。日本語なら日本語で、スペイン語ならスペイン語で応答してください。

---

## Claudeへの指示

マーケティングプロジェクトのためのファイル整理、コマンド使用、ドキュメント参照を教えてください。

### レッスン概要

---

**モジュール1.2: マーケティングファイルの操作**

マーケターとして、キャンペーンブリーフ、コンテンツドラフト、リサーチドキュメント、分析レポートなど多くのアセットタイプを扱います。効率的に整理・管理する方法をマスターしましょう。

**所要時間:** 約25分

---

### ステップ1: ドキュメント構造をレビューする

docsフォルダを見せてください：

```
docs/内のすべてのファイルをリストする
```

各ドキュメントファイルを説明してください：
- `brand-guidelines.md` - ブランド基準テンプレート
- `content-style-guide.md` - ライティング基準、CTA、フォーマット
- `campaign-playbooks.md` - 実証済みのキャンペーンテンプレート
- `channel-strategies.md` - プラットフォーム別戦術
- `analytics-setup.md` - トラッキングとアトリビューション
- `usage-guide.md` - 完全なシステムリファレンス

### ステップ2: キャンペーンプレイブックを探索する

キャンペーンプレイブックを読んでください：

```
docs/campaign-playbooks.mdを読む
```

プレイブックタイプを説明してください：
- 製品ローンチプレイブック
- リード生成プレイブック
- ブランド認知プレイブック
- リテンションプレイブック
- イベントプロモーションプレイブック

### ステップ3: コンテンツコマンドを練習する

コンテンツ作成コマンドをガイドしてください：

**ブログ投稿：**
```
/content:blog "リモートチームが調整を改善する5つの方法" "リモートチームの生産性"
```

**ソーシャルコンテンツ：**
```
/content:social "リモートマネージャーのためのチーム調整のヒント" "linkedin"
```

**メールコピー：**
```
/content:email "welcome" "AgentKitsのトライアルユーザー"
```

### ステップ4: 検索コマンドを練習する

grep/findまたはClaudeに質問して検索テクニックを教えてください：

```
「リードスコアリング」について言及しているすべてのファイルを見つける
```

```
「コンバージョン率」を含むファイルを検索する
```

### ステップ5: バッチコンテンツ作成

一度に複数のアセットを作成することを示してください：

```
AgentKitsローンチ用のマルチチャネルコンテンツを作成：
1. LinkedIn発表投稿
2. Twitterスレッド（5ツイート）
3. メール件名（5つのA/Bバリエーション）
4. Google Ads見出し（5バリエーション、最大30文字）
```

### ステップ6: スタイルガイドとのクロスリファレンス

コンテンツスタイルガイドの使い方を示してください：

```
docs/content-style-guide.mdを読む
```

指摘してください：
- 見出しの公式（4-Uフレームワーク）
- CTAパターン
- 可読性基準
- SEOライティングガイドライン

### ステップ7: クイックリファレンスコマンド

重要なコマンドパターンを共有してください：

**キャンペーンコマンド：**
- `/campaign:plan` - キャンペーン計画を作成
- `/campaign:brief` - クリエイティブブリーフを生成
- `/campaign:analyze` - パフォーマンスを分析
- `/campaign:calendar` - コンテンツカレンダー

**コンテンツコマンド：**
- `/content:blog` - SEOブログ投稿
- `/content:social` - プラットフォーム別ソーシャル
- `/content:email` - メールコピー
- `/content:landing` - ランディングページコピー
- `/content:ads` - 広告コピー

- 6つの主要ドキュメントがブランド、コンテンツ、キャンペーン、チャネル、分析、使用法をカバー

- 一貫性のためにドキュメントをクロスリファレンス
- バッチ操作で大幅に時間を節約
