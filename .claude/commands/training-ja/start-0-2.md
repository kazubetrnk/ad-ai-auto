# /training:start-0-2 - 最初のマーケティングタスク

## 言語と品質基準

**重要**: ユーザーが使用している言語で応答してください。日本語なら日本語で、スペイン語ならスペイン語で応答してください。

---

## Claudeへの指示

学生が最初の実際のマーケティングタスク - コマンドを使用してブランドガイドラインを作成するようにガイドしてください。

### レッスン概要

---

**モジュール0.2: 最初のマーケティングタスク**

では実際のマーケティング作業を行いましょう。`/brand:voice`コマンドを使用してAgentKitsのブランドガイドラインを作成します。

---

### ステップ1: ブランドガイドラインを説明する

ブランドガイドラインが重要な理由を説明してください：
- すべてのコンテンツで一貫性を確保
- Claude（と人間）が正しい声で書くのを助ける
- 重要なメッセージと用語を文書化
- ブランド外のコンテンツを防ぐ

### ステップ2: Brand Voiceコマンドを使用する

実際のシステムコマンドを使用するようにガイドしてください：

```
/brand:voice "AgentKits - リモートチーム向けB2Bチーム生産性調整ツール"
```

Claudeに包括的なブランドガイドラインを生成させ、学生とレビューしてください。

### ステップ3: 顧客ペルソナを作成する

ペルソナのためにリサーチコマンドを使用してください：

```
/research:persona "AgentKitsを使用するテクノロジー企業のリモートチームマネージャー"
```

説明してください：
- `researcher`エージェントが市場調査を処理します
- ペルソナは特定のオーディエンスにコンテンツをターゲットするのに役立ちます
- コース全体を通してこれらのペルソナを使用します

### ステップ4: 作成されたものをレビューする

一緒に出力をレビューしてください：

```
brand:voiceコマンドが作成したものを見せてください
```

指摘してください：
- 声の属性とトーンスペクトル
- メッセージングフレームワーク
- 使用すべき言葉と避けるべき言葉
- 他のエージェントとの統合

### ステップ5: コンテキストの力

説明してください：
- これらのガイドラインはプロジェクトコンテキストに存在します
- 将来のタスクでエージェントがそれらを参照できます
- これが「コンテキスト認識」- Claudeのスーパーパワーの1つです
- コース全体を通してこれらを使用します

### クイック演習: コンテンツ生成をテストする

ブランドコンテキストを使用してもらってください：

```
/content:social "リモートチームの生産性のヒント" "linkedin"
```

Claudeがコンテンツ作成でブランドコンテキストを自動的に使用することを示してください。

### ステップ6: 他の主要コマンドを探索する

マスターする他のコマンドを簡単にデモンストレーションしてください：

**キャンペーン計画：**
```
/campaign:plan "Q1製品ローンチ"
```

**SEOリサーチ：**
```
/seo:keywords "リモートチームの生産性"
```

**メールシーケンス：**
```
/sequence:welcome "AgentKits" "トライアルユーザー"
```

## 主な教育ポイント
- ブランドガイドラインは一貫性を確保します
- `/brand:voice`を使用して声のガイドラインを作成します
- `/research:persona`を顧客ペルソナに使用します
- コンテキスト認識はエージェントが既存の作業を参照することを意味します
- 実際のコマンド（`/campaign:*`、`/content:*`、`/seo:*`）がマスターするものです
