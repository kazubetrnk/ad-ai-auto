# /training:start-2-2 - コンテンツ戦略を開発する

## 言語と品質基準

**重要**: ユーザーが使用している言語で応答してください。日本語なら日本語で、スペイン語ならスペイン語で応答してください。

---

## Claudeへの指示

包括的なコンテンツ戦略開発を教えてください：リサーチ、計画、カレンダー、測定。

### レッスン概要

---

**モジュール2.2: コンテンツ戦略を開発する**

コンテンツ戦略はランダムなコンテンツ作成を体系的な成長エンジンに変えます。AgentKits用に構築しましょう。

**所要時間:** 約40分

---

### ステップ1: リサーチの基盤

市場とオーディエンスリサーチから始めてください：

```
/research:market "B2Bチーム生産性ソフトウェア - リモートワークツール市場"
```

```
/research:persona "テクノロジー企業のリモートチームマネージャー - 50-500人の従業員"
```

```
/research:trend "リモートワーク生産性 - チーム調整 - 非同期ワーク"
```

### ステップ2: SEOキーワードリサーチ

キーワードの基盤のためにSEOコマンドを使用してください：

```
/seo:keywords "リモートチームの生産性"
```

```
/seo:keywords "チーム調整ソフトウェア"
```

```
/seo:keywords "チームのためのディープワーク"
```

キーワードをトピッククラスターにグループ化してください：
- クラスター1: リモートチームの生産性
- クラスター2: チームフォーカスタイム
- クラスター3: 会議なしの調整

### ステップ3: 競合コンテンツ分析

競合コンテンツを分析してください：

```
/seo:competitor "rescuetime.com"
```

特定してください：
- カバーしているトピック
- 埋められるコンテンツギャップ
- ランキングしているキーワード

### ステップ4: コンテンツカレンダーを作成する

コンテンツ計画のためにキャンペーンカレンダーを使用してください：

```
/campaign:calendar "12週間 - AgentKitsコンテンツ戦略 - SEO、ソートリーダーシップ、リード生成に焦点 - トピック: リモート生産性、チーム調整、ディープワーク、会議削減"
```

### ステップ5: コンテンツタイプを定義する

ファネルステージ別にコンテンツを計画してください：

**TOFU（認知）：**
- ブログ投稿（SEO重視）
- ソーシャルメディアコンテンツ
- ソートリーダーシップ

**MOFU（検討）：**
- 比較ガイド
- ハウツーコンテンツ
- ケーススタディ

**BOFU（決定）：**
- 製品デモ
- ROI計算機
- 顧客の声

### ステップ6: ピラーコンテンツ戦略を作成する

ピラーページ戦略を計画してください：

```
/content:blog "リモートチーム生産性完全ガイド：タイムゾーン間でフォーカスタイムを調整する方法" "リモートチームの生産性"
```

クラスターコンテンツ（ピラーにリンク）：
1. チームフォーカスタイムのスケジュール方法
2. アライメントを失わずに会議を減らす
3. 非同期コミュニケーションのベストプラクティス
4. リモートチームでのディープワーク
5. チームの生産性トラッキング

### ステップ7: コンテンツ制作ワークフロー

各ピースにコンテンツコマンドを使用してください：

**ブログ投稿制作：**
```
1. /seo:keywords "トピック" - キーワードリサーチ
2. /content:blog "タイトル" "キーワード" - 投稿を作成
3. /seo:optimize "投稿" "キーワード" - 最適化
4. seo-specialistエージェントでレビュー
5. brand-voice-guardianエージェントでレビュー
```

**ソーシャルコンテンツ制作：**
```
1. /content:social "トピック" "linkedin" - LinkedIn投稿
2. /content:social "トピック" "twitter" - Twitterスレッド
3. conversion-optimizerエージェントでレビュー
```

### ステップ8: メール統合

コンテンツ消費者をナーチャーするメールシーケンスを作成してください：

```
/sequence:nurture "AgentKits" "ガイドをダウンロードしたブログ読者"
```

### ステップ9: コンテンツ配信計画

配信のためにソーシャルコマンドを使用してください：

```
/social:schedule "linkedin,twitter" "4週間 - AgentKitsコンテンツ配信"
```

### ステップ10: 測定フレームワーク

コンテンツ分析をセットアップしてください：

```
/analytics:report "コンテンツパフォーマンス" "オーガニックトラフィック、エンゲージメント、コンバージョン"
```

追跡する主要メトリクス：
- コンテンツピース別オーガニックトラフィック
- ページ滞在時間
- コンテンツ別コンバージョン率
- コンテンツからのリード品質

## 主な教育ポイント
- 戦略がコンテンツをランダムから体系的に変える
- `/research:*`コマンドが基盤を構築
- `/seo:keywords`が機会を特定
- ピラー + クラスター = SEOパワーハウス
- コンテンツ制作は繰り返し可能なワークフローに従う
- 測定がアカウンタビリティを確保
