# /training:start-1-3 - 最初のマーケティングタスク

## 言語と品質基準

**重要**: ユーザーが使用している言語で応答してください。日本語なら日本語で、スペイン語ならスペイン語で応答してください。

---

## Claudeへの指示

実際のシステムコマンドを使用して、マルチチャネルコピー、競合分析、コンテンツ計画などの実際のマーケティングタスクを学生にガイドしてください。

### レッスン概要

---

**モジュール1.3: 最初のマーケティングタスク**

では実際のマーケティング作業を行いましょう。すべてのマーケターが定期的に行う3つの一般的なタスクを完了します。

**所要時間:** 約30分

---

### タスク1: マルチチャネルコピー生成

コンテンツコマンドを使用して複数のチャネル用のコピーを生成：

**LinkedIn投稿：**
```
/content:social "AgentKitsローンチ発表 - リモートチームのためのチーム生産性調整" "linkedin"
```

**ブログ投稿：**
```
/content:blog "リモートチームが終わりのない会議なしにフォーカスタイムを調整する方法" "リモートチーム調整"
```

**メール：**
```
/content:email "製品発表" "既存の購読者"
```

一緒に出力をレビューしてください。イテレーションを示してください：

```
LinkedInの投稿をより教育的に - ディープワーク時間の調整問題に焦点を当てて
```

### タスク2: 競合分析

競合分析コマンドを使用してください：

```
/competitor:deep "RescueTime - 個人の時間追跡と生産性ソフトウェア"
```

`researcher`エージェントが分析する内容を説明してください：
- ターゲットオーディエンス
- 主要機能とポジショニング
- 価格モデル
- 強みと弱み
- 市場機会

フォローアップを質問してください：
```
この分析に基づいて、AgentKitsの最大の競争優位性は何ですか？
```

### タスク3: コンテンツカレンダー

キャンペーンカレンダーコマンドを使用してください：

```
/campaign:calendar "4週間 - AgentKits製品ローンチ - リモートワーク生産性、チームコラボレーション、ディープワークに焦点"
```

生成されたカレンダーをレビューしてください：
- SEOキーワード付きブログ投稿トピック
- プラットフォーム別ソーシャルメディアテーマ
- メールニュースレタースケジュール
- 各コンテンツのコンテンツ目標

### ステップ4: 1つを展開する

トピックを取り上げてコンテンツコマンドで展開してください：

```
/content:blog "チームフォーカスタイム完全ガイド：リモートチームがディープワークを調整する方法" "チームフォーカスタイム"
```

### ステップ5: SEO最適化

SEOコマンドを使用して最適化してください：

```
/seo:keywords "リモートチームの生産性"
```

次に：
```
/seo:optimize "作成したばかりのブログ投稿" "チームフォーカスタイム"
```

### ステップ6: スペシャリストとレビュー

レビュアーエージェントを使用してください（これらは後で詳しく説明されます）：

```
3つの視点からブログ投稿をレビューしてください：
1. Brand Voice Guardian - 私たちの声に合っていますか？
2. SEO Specialist - 検索に最適化されていますか？
3. Conversion Optimizer - アクションを促しますか？
```

### 祝う

達成したことを指摘してください：
- `/content:*`コマンドを使用したマルチチャネルコピー生成
- `/competitor:deep`を使用した競合分析
- `/campaign:calendar`を使用したコンテンツカレンダー
- `/seo:keywords`を使用したSEOキーワードリサーチ
- SEO最適化付きの完全なブログ投稿

## 主な教育ポイント
- 実際のコマンドが実際のマーケティングタスクを処理します
- `/content:*`コマンドがプラットフォーム別コンテンツを作成します
- `/competitor:deep`が競合インテリジェンスを提供します
- `/campaign:calendar`がコンテンツカレンダーを作成します
- `/seo:*`コマンドが検索最適化を処理します
- 常にコンテキストを提供してください（ブランド、オーディエンス、目標）
