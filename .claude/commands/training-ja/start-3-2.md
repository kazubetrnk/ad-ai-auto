# モジュール3、レッスン2：フォームとサインアップの最適化

## フォームと登録CROをマスターする

フォームはコンバージョンゲートキーパーです。不必要なフィールドごとにリードを失います。このレッスンでは、リードキャプチャフォームとサインアップフローを最適化する方法を教えます。

## 学習目標

このレッスンの終わりまでに：
1. 5フィールド最大ルールを適用する
2. コンバージョン向けにサインアップフローを最適化
3. フォームフリクションを体系的に削減
4. マルチステッププログレッシブフォームをデザイン

---

## フォームCRO原則

### 5フィールドルール

5を超える各フィールドはコンバージョンを約10%減少させます。

**必須フィールドのみ：**
1. メール（常に必須）
2. 名前（時々）
3. 会社（B2Bのみ）
4. パスワード（サインアップのみ）
5. 必要に応じて1つの資格確認

**他のすべて**はサインアップ後に延期。

### 排除すべきフリクションポイント

| フリクション | 修正 |
|----------|-----|
| フィールドが多すぎる | 削除または延期 |
| パスワード要件 | エラー後ではなくインラインで表示 |
| 必須電話 | オプションにするか削除 |
| CAPTCHA | 見えない代替手段を使用 |
| ソーシャルログインなし | Google/SSOオプションを追加 |

---

## フォームCROコマンド

リードキャプチャフォームには`/cro:form`を使用：

```bash
/cro:form "AgentKitsのデモリクエストフォームを最適化：名前、メール、会社、電話、役職、チームサイズ、メッセージ"
```

### 期待される推奨事項

1. **削除：** メッセージ（フォローアップで質問）
2. **削除：** 電話（後でキャプチャ可能）
3. **オプション化：** 役職
4. **維持：** 名前、メール、会社、チームサイズ

7 → 4フィールドに削減 = 推定+30%コンバージョン

---

## サインアップフロー最適化

アカウント登録には`/cro:signup`を使用：

```bash
/cro:signup "AgentKitsのトライアルサインアップを分析：メール → パスワード → 会社 → チームサイズ → ユースケース → 支払い"
```

### サインアップフローパターン

| パターン | 最適な用途 | コンバージョン |
|---------|----------|------------|
| メールのみ開始 | 最高のコンバージョン | メールで開始、プログレッシブプロファイリング |
| ソーシャルファースト | コンシューマーアプリ | Google/SSOを目立たせる |
| シングルページミニマル | B2C、シンプルな製品 | すべてのフィールドが見える |
| プログレス付きマルチステップ | B2B、複雑 | ガイド付き、プログレスを表示 |

## 演習1：フォーム監査

AgentKitsの現在のデモリクエストフォームを監査：

```bash
/cro:form "7フィールドフォーム：名前、メール、会社、電話、役職、チームサイズ、メッセージ。目標：デモコールをスケジュール。"
```

推奨事項を作成：
```
training/exercises/markit/cro/form-audit.md
```

### 出力に含めるべきもの

1. 削除するフィールド
2. オプションにするフィールド
3. プログレッシブプロファイリング戦略
4. コピー改善（ボタンテキスト、ラベル）

---

## 演習2：サインアップフローの再設計

AgentKits用の最適化されたサインアップフローをデザイン：

```bash
/cro:signup "AgentKitsに最適なトライアルサインアップをデザイン。現在：5ステッププロセス。目標：トライアル開始を最大化。"
```

考慮事項：
- メールのみの初期キャプチャ
- OAuthオプション（B2B向けGoogle Workspace）
- 会社情報をいつ収集するか
- オンボーディングvsサインアップの分離

---

## 演習3：マルチステップフォームデザイン

複雑なB2Bサインアップには、マルチステップアプローチをデザイン：

**ステップ1：** メールのみ
**ステップ2：** 会社 + チームサイズ（プログレスバー付き）
**ステップ3：** オプションのユースケース選択

form-croスキルを使用して検証：

```bash
/cro:form "AgentKitsエンタープライズデモ用の3ステッププログレッシブフォームをデザイン"
```
