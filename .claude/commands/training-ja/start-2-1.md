# /training:start-2-1 - キャンペーンブリーフを書く

## 言語と品質基準

**重要**: ユーザーが使用している言語で応答してください。日本語なら日本語で、スペイン語ならスペイン語で応答してください。

---

## Claudeへの指示

モジュール2 - 高度なアプリケーションを開始します。このレッスンではキャンペーンコマンドを使用した包括的なキャンペーンブリーフ作成を教えます。

### レッスン概要

---

**モジュール2.1: キャンペーンブリーフを書く**

モジュール2へようこそ！学んだすべてを実際のマーケティングワークフローに適用します。キャンペーンブリーフは成功した実行の基盤です。

**所要時間:** 約45分

---

### ステップ1: コラボレーティブアプローチを説明する

> Claudeはあなたの戦略的パートナーであり、マーケティング専門知識の代替ではありません。あなたがインサイト、市場知識、戦略的思考をもたらします。Claudeはそれらのアイデアを表現し構造化するのを助けます。

### ステップ2: 戦略的インプットを収集する

学生に戦略的思考を尋ねてください：

```
AgentKitsのQ2成長キャンペーンのための包括的なキャンペーンブリーフを作成しましょう。

まず、あなたの戦略的思考を教えてください：
- 主要な目標は何ですか？（例：2000トライアルサインアップ）
- 予算は？
- 期間は？
- 注力すべき特定のチャネルはありますか？
- この四半期の重要なメッセージは？
```

入力を待ってから続行してください。

### ステップ3: キャンペーンプランコマンドを使用する

キャンペーン計画コマンドを使用してください：

```
/campaign:plan "AgentKits Q2成長 - 目標: 2000トライアルサインアップ、予算: 7万5千ドル、期間: 8週間、チャネル: LinkedIn Ads、コンテンツマーケティング、メールナーチャー、主要メッセージ: チーム全体のフォーカスタイム調整"
```

`planner`エージェントによって生成された包括的なキャンペーン計画をレビューしてください。

### ステップ4: クリエイティブブリーフを生成する

クリエイティブブリーフコマンドを使用してください：

```
/campaign:brief "AgentKits Q2成長キャンペーン"
```

クリエイティブブリーフに含まれるものを説明してください：
- シングルマインドプロポジション
- ターゲットオーディエンスインサイト
- トーンとマナー
- デリバラブルリスト
- クリエイティブマンダトリー

### ステップ5: マルチパースペクティブフィードバックを取得する

レビュアーエージェントを使用してください：

```
3つの視点からQ2キャンペーン計画をレビューしてください：

1. `manager-maria`（マーケティングマネージャー）- マーケティングチームが実行可能ですか？
2. `conversion-optimizer` - このキャンペーン構造はコンバージョンを促進しますか？
3. `brand-voice-guardian` - メッセージングはオンブランドですか？

具体的なフィードバックと推奨事項を提供してください。
```

### ステップ6: コンテンツカレンダーを作成する

カレンダーコマンドを使用してください：

```
/campaign:calendar "8週間 - AgentKits Q2成長 - コンテンツマーケティング、ソーシャルメディア、メールナーチャーをチーム生産性に焦点"
```

### ステップ7: サポートドキュメントを作成する

追加のキャンペーンアセットを生成してください：

**リードスコアリングモデル：**
```
/leads:score "B2B SaaS - テクノロジー企業 - チーム生産性"
```

**ウェルカムシーケンス：**
```
/sequence:welcome "AgentKits" "Q2キャンペーンからのトライアルサインアップ"
```

**ナーチャーシーケンス：**
```
/sequence:nurture "AgentKits" "コンバートしていないエンゲージドリード"
```

### ステップ8: 競合準備

競合資料を準備してください：

```
/sales:battlecard "RescueTime - 生産性ツールの主な競合"
```

```
/competitor:deep "Freedomアプリ - フォーカスと生産性ブロッキング"
```

### ステップ9: 測定計画を作成する

分析をセットアップしてください：

```
/analytics:funnel "トライアルサインアップファネル - 認知からトライアルから有料まで"
```

### ステップ10: テンプレートとして保存する

このワークフローはどのキャンペーンにも繰り返し使用できることを説明してください：

```
キャンペーンブリーフワークフロー：
1. /campaign:plan - 戦略計画
2. /campaign:brief - クリエイティブブリーフ
3. /campaign:calendar - コンテンツカレンダー
4. /leads:score - リード資格
5. /sequence:welcome - 新規リードナーチャー
6. /sequence:nurture - 継続的ナーチャー
7. /sales:battlecard - 競合準備
8. /analytics:funnel - 測定セットアップ
```

### 次は

伝えてください：
- 1時間以内にプロフェッショナルなキャンペーンブリーフを作成しました
- 通常これは複数のミーティングで数日かかります
- **次:** `/training:start-2-2` - コンテンツ戦略を開発する
- 包括的なコンテンツ計画を構築します

## 主な教育ポイント
- キャンペーンブリーフはコラボレーティブ
- `/campaign:plan`を戦略計画に使用
- `/campaign:brief`をクリエイティブディレクションに使用
- `/campaign:calendar`をコンテンツスケジューリングに使用
- フィードバックにレビュアーエージェントを使用
- サポートアセットを作成（リードスコアリング、シーケンス、バトルカード）
