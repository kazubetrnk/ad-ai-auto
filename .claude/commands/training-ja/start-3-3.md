# モジュール3、レッスン3：ポップアップとオンボーディングCRO

## 訪問者のコンバージョンとユーザーのアクティベーション

このレッスンでは、2つの重要なコンバージョンポイントをカバーします：ポップアップで訪問者をキャプチャし、オンボーディングで新規サインアップをアクティベートします。

## 学習目標

このレッスンの終わりまでに：
1. ユーザーを煩わせずに高コンバージョンのポップアップをデザインする
2. サインアップ後のオンボーディングフローを最適化
3. 「Ahaモーメント」を特定して加速
4. ペイウォールとアップグレード画面を作成

---

## ポップアップCRO

### ポップアップが機能するとき

| タイプ | トリガー | 最適な用途 |
|------|---------|----------|
| 離脱インテント | マウスがビューポートを離れる | リードキャプチャ、離脱者を救う |
| 時間遅延 | ページで30-60秒 | エンゲージした訪問者 |
| スクロールトリガー | 50-70%スクロール深度 | コンテンツエンゲージメント |
| クリックトリガー | ユーザーアクション | 特定のCTA |

### ポップアップが失敗するとき

- ページロード時に即座に表示
- 明確なバリュープロポジションがない
- 閉じにくい
- 毎回同じポップアップ

---

## ポップアップデザイン演習

`/cro:popup`を使用して効果的なポップアップをデザイン：

```bash
/cro:popup "AgentKitsブログ用の離脱インテントポップアップをデザイン。目標：'Remote Team Productivity Guide'リードマグネット用のメールをキャプチャ。"
```

### 良いポップアップ要素

1. **明確な価値：** 何を得られるか
2. **最小限のフィールド：** メールのみ
3. **簡単な閉じる：** 見える×
4. **モバイルフレンドリー：** 親指が届くCTA
5. **頻度キャップ：** セッションごとに1回

---

## オンボーディングCRO

### アクティベーション方程式

**Ahaモーメント** = ユーザーが初めてコアバリューを体験する時

AgentKitsの場合：「チームメンバーがチームのフォーカススケジュールを見て、邪魔されない時間をブロックするとき」

### オンボーディングパターン

| パターン | 最適な用途 |
|---------|----------|
| セットアップウィザード | 設定が必要な複雑な製品 |
| チェックリスト | 機能豊富なアプリ |
| インタラクティブツアー | UI重視の製品 |
| テンプレートギャラリー | クリエイティブツール |
| サンプルプロジェクト | プロジェクトベースのツール |

---

## オンボーディング演習

`/cro:onboarding`を使用してAgentKitsのアクティベーションを最適化：

```bash
/cro:onboarding "AgentKits用のオンボーディングをデザイン。Ahaモーメント：チームフォーカススケジュールを見る。現在のアクティベーション：トライアルの15%。目標：40%。"
```

### 主要な質問

1. 価値を得るための最小限のセットアップは？
2. セットアップ前に価値を見せられるか？
3. コンバージョンを予測する#1アクションは？
4. ユーザーはどれくらい速くAhaモーメントに到達できるか？

---

## ペイウォールとアップグレードCRO

フリーミアムとトライアル製品では、アップグレード画面が重要です。

### ペイウォールトリガー

| トリガー | コンテキスト |
|---------|---------|
| 機能ゲート | ユーザーがプレミアム機能を試す |
| 使用制限 | 無料ティアの制限に到達 |
| トライアル期限 | 時間ベースのトライアル終了 |
| アップグレードプロンプト | 価値の瞬間の後 |

### ペイウォール演習

```bash
/cro:paywall "AgentKits用のアップグレード画面をデザイン。トリガー：ユーザーが無料ティアで5ユーザー制限に到達。目標：Teamプラン（$12/ユーザー）にコンバート。"
```

## フルCROファネル

これで完全なコンバージョンファネルを最適化できます：

```
訪問者 → Page CRO → Form CRO → Signup CRO
     ↓
  Popup CRO（離脱者をキャプチャ）
     ↓
新規ユーザー → Onboarding CRO → アクティベーション
     ↓
無料ユーザー → Paywall CRO → 有料顧客
```

各スキルが特定のステージを処理します。
