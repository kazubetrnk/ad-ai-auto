### レッスン概要

---

**モジュール1.5: レビュアーエージェント詳解**

マーケティングキットには6つの専門レビュアーエージェントが含まれています。これらは異なる視点からエキスパートレベルのフィードバックを提供し、コンテンツ品質を確保します。

**所要時間:** 約30分

---

### ステップ1: なぜレビュアーエージェントか？

すべてのマーケティングは複数の角度からレビューされるべきです：
- ブランドの一貫性
- SEO最適化
- コンバージョンポテンシャル
- ペルソナフィット
- 実践的な実装

レビュアーエージェントは各領域のオンデマンドエキスパートを提供します。

### ステップ2: エキスパートレビュアー

3つのエキスパートレビュアーを説明してください：

**`brand-voice-guardian`：**
- ブランドボイスの一貫性を確保
- トーンとメッセージングをチェック
- ブランドガイドラインに対して検証
- オフブランドコンテンツにフラグ

**`seo-specialist`：**
- キーワード使用を評価
- オンページSEO要素をチェック
- コンテンツ構造をレビュー
- 最適化の機会を特定

**`conversion-optimizer`：**
- CTAの効果を評価
- 説得要素をレビュー
- フリクションポイントを特定
- コンバージョン改善を提案

### ステップ3: ペルソナレビュアー

3つのペルソナレビュアーを説明してください：

**`manager-maria`（マーケティングマネージャー、38歳）：**
- B2B中規模企業の視点
- チーム実装フィットを評価
- リソース制約を考慮
- キャンペーンの実現可能性をレビュー

**`solo-steve`（個人事業主、32歳）：**
- 一人ビジネスの視点
- 時間効率を評価
- セルフサービスの容易さをレビュー
- 予算感度を考慮

**`startup-sam`（スタートアップ創業者、28歳）：**
- アーリーステージスタートアップの視点
- 成長ポテンシャルを評価
- バイラル性要素をレビュー
- 市場投入スピードを評価

### ステップ4: エキスパートレビュアーを使用する

エキスパートレビュアーで練習してください：

```
エキスパートレビュアーでAgentKitsブログ投稿をレビューしてください：

1. `brand-voice-guardian`：ブランドの一貫性を評価
2. `seo-specialist`：SEO最適化を評価
3. `conversion-optimizer`：コンバージョンポテンシャルを評価

各レビュアーは（1-10）でスコアリングし、以下を提供：
- トップ3の強み
- トップ3の必要な改善
- 具体的な推奨編集
```

### ステップ5: ペルソナレビュアーを使用する

ペルソナレビュアーで練習してください：

```
ペルソナの視点からAgentKitsメールシーケンスをレビューしてください：

1. `manager-maria`：B2Bマーケティングマネージャーはこれを価値あると思うか？
2. `solo-steve`：個人事業主はこれを実装するか？
3. `startup-sam`：これはスタートアップの成長を促進するか？

各レビュアーは評価してください：
- ニーズとの共鳴
- 実装の実現可能性
- 価値の認識
```

### ステップ6: 統合レビューボード

包括的なフィードバックのためにすべてのレビュアーを一緒に使用してください：

```
AgentKitsランディングページコピーでフルレビューボードを実行してください：

エキスパートレビュー：
- ブランドの一貫性（brand-voice-guardian）
- SEO最適化（seo-specialist）
- コンバージョンポテンシャル（conversion-optimizer）

ペルソナレビュー：
- B2Bフィット（manager-maria）
- セルフサービスポテンシャル（solo-steve）
- 成長ポテンシャル（startup-sam）

統合：すべてのレビュアーを満足させる変更は何ですか？
```

### ステップ7: イテレーティブな改善

レビュー-改善サイクルを示してください：

```
1. コンテンツを作成
2. マルチレビュアーフィードバックを取得
3. 最優先の変更を実装
4. 変更されたコンテンツを再レビュー
5. すべてのレビュアーが8+のスコアになるまで繰り返す
```

### ステップ8: どのレビュアーをいつ使用するか

ガイダンスを提供してください：

| コンテンツタイプ | 主要レビュアー |
|-------------|-------------------|
| ブログ投稿 | seo-specialist、brand-voice-guardian |
| ランディングページ | conversion-optimizer、manager-maria |
| メールシーケンス | email-wizard、startup-sam |
| ソーシャルコンテンツ | brand-voice-guardian、solo-steve |
| セールス資料 | sales-enabler、manager-maria |

## 主な教育ポイント
- 6つのレビュアーエージェント：3つのエキスパート、3つのペルソナ
- エキスパートレビュアーは品質次元をチェック
- ペルソナレビュアーはオーディエンスフィットをチェック
- 包括的なフィードバックには複数のレビュアーを使用
- レビューサイクルを通じてコンテンツを反復
