### レッスン概要

---

**モジュール2.6: SEO最適化**

SEOは持続可能で長期的な成長を促進します。キーワードリサーチ、コンテンツ最適化、テクニカルSEOをマスターしましょう。

**所要時間:** 約40分

---

### ステップ1: キーワードリサーチ

キーワードリサーチコマンドを使用してください：

```
/seo:keywords "リモートチームの生産性"
```

```
/seo:keywords "チームフォーカスタイムソフトウェア"
```

```
/seo:keywords "リモートチームの調整ツール"
```

キーワード機会をレビューしてください：
- 検索ボリューム推定
- 競合レベル
- インテント分類
- キーワードクラスター

### ステップ2: 競合SEO分析

競合SEOを分析してください：

```
/seo:competitor "rescuetime.com"
```

```
/seo:competitor "freedom.to"
```

特定してください：
- ランキングしているキーワード
- コンテンツギャップ
- バックリンク機会
- テクニカルの強み/弱み

### ステップ3: SEO監査

包括的な監査を実行してください：

```
/seo:audit "agentkits.com"
```

監査領域をレビューしてください：
- テクニカルSEO（クロール可能性、速度、モバイル）
- オンページSEO（タイトル、メタ、ヘッダー）
- コンテンツ品質（深さ、関連性）
- バックリンクプロファイル

### ステップ4: コンテンツ最適化

既存のコンテンツを最適化してください：

```
/seo:optimize "チーム生産性についてのブログ投稿" "リモートチームの生産性"
```

最適化に含まれるもの：
- タイトルタグの改善
- メタディスクリプション
- ヘッダー構造（H1、H2、H3）
- キーワード配置
- 内部リンク提案
- 画像altテキスト

### ステップ5: SEOコンテンツを作成する

SEO重視でブログコマンドを使用してください：

```
/content:blog "リモートチーム生産性完全ガイド" "リモートチームの生産性"
```

次に最適化：

```
/seo:optimize "ブログ投稿" "リモートチームの生産性"
```

### ステップ6: フィーチャードスニペット戦略

フィーチャードスニペット向けに最適化してください：

```
フィーチャードスニペット向けに最適化されたコンテンツを作成：

ターゲットクエリ：「リモートチームの生産性を改善する方法」

パラグラフスニペット（40-60語）：
直接回答形式

リストスニペット（番号付きステップ）：
ステップバイステップ形式

テーブルスニペット：
比較形式
```

### ステップ7: テクニカルSEOチェックリスト

SEOチェックリストを使用してください：

```
/checklist:seo-weekly "agentkits.com"
```

週次SEOタスク：
- クロールエラーチェック
- ページ速度モニタリング
- モバイルユーザビリティ
- インデックスステータス
- キーワードトラッキング

### ステップ8: ピラー-クラスター戦略

ピラーコンテンツ計画を作成してください：

```
ピラー-クラスターSEO戦略を作成：

ピラーページ：「リモートチーム生産性完全ガイド」
- 3000-5000語、包括的

クラスターコンテンツ（10投稿）：
1. 「チームフォーカスタイムのスケジュール方法」 → チームフォーカスタイム
2. 「アライメントを失わずに会議を減らす」 → 会議削減リモート
3. 「非同期コミュニケーションのベストプラクティス」 → 非同期コミュニケーション
4. 「リモートチームのディープワーク」 → ディープワークチーム
5. 「重要なチーム生産性メトリクス」 → 生産性メトリクス
...

内部リンク：
- 各クラスターはピラーにリンク
- ピラーはすべてのクラスターにリンク
- 関連クラスター間のクロスリンク
```

### ステップ9: SEO付きコンテンツカレンダー

SEO重視のカレンダーを作成してください：

```
/campaign:calendar "12週間 - SEOコンテンツ戦略 - ターゲットキーワード: リモートチームの生産性、チームフォーカスタイム、調整ソフトウェア"
```

コンテンツをキーワードにマッピング：
- 週1：[キーワード] - [タイトル] - [タイプ]
- 週2：[キーワード] - [タイトル] - [タイプ]
...

### ステップ10: SEO測定

SEOパフォーマンスを追跡してください：

```
/analytics:report "SEOパフォーマンス" "オーガニックトラフィック、キーワードランキング、コンバージョン"
```

主要メトリクス：
- オーガニックトラフィックトレンド
- キーワード順位変動
- オーガニックコンバージョン率
- ページパフォーマンス
