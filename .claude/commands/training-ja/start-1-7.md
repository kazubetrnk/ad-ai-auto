### ステップ2: コマンドカテゴリリファレンス

すべてのコマンドカテゴリをレビューしてください：

| カテゴリ | コマンド | ユースケース |
|----------|----------|-----------|
| `/campaign:*` | plan、brief、analyze、calendar | キャンペーン管理 |
| `/content:*` | blog、social、email、landing、ads | コンテンツ作成 |
| `/seo:*` | keywords、audit、optimize、competitor | SEO最適化 |
| `/analytics:*` | roi、funnel、report | パフォーマンス追跡 |
| `/sales:*` | pitch、outreach、battlecard、qualify | セールス支援 |
| `/leads:*` | score、nurture、qualify | リード管理 |
| `/crm:*` | sequence、segment、score、lifecycle | CRM自動化 |
| `/sequence:*` | welcome、nurture、re-engage | メールシーケンス |
| `/social:*` | engage、viral、schedule | ソーシャルメディア |
| `/research:*` | market、persona、trend | 市場調査 |
| `/brand:*` | voice、book、assets | ブランド管理 |
| `/ops:*` | daily、weekly、monthly | オペレーション |
| `/report:*` | weekly、monthly | レポート |
| `/checklist:*` | campaign-launch、seo-weekly | チェックリスト |

### ステップ5: クイックリファレンスコマンド

重要なコマンドショートカットを共有してください：

**高速コンテンツ：**
```
/content:fast "ソーシャル投稿のクイックヘッドライン"
```

**良質コンテンツ：**
```
/content:good "リサーチ付きの詳細なブログ投稿"
```

**ブレインストーミング：**
```
/brainstorm "B2B SaaSのマーケティングチャネルアイデア"
```

### ステップ6: オペレーショナルコマンド

オペレーショナルワークフローを練習してください：

**日次オペレーション：**
```
/ops:daily "コンテンツ作成"
```

**週次レビュー：**
```
/ops:weekly "今週の日付"
```

**キャンペーンチェックリスト：**
```
/checklist:campaign-launch "Q1製品ローンチ" "来週"
```

### ステップ7: 完全なコマンドワークフロー

完全なワークフローをデモンストレーションしてください：

```
1. /research:market "B2B生産性ソフトウェア"
2. /campaign:plan "Q2成長キャンペーン"
3. /seo:keywords "チームの生産性"
4. /content:blog "生産性トピック" "ターゲットキーワード"
5. /seo:optimize "ブログ投稿" "キーワード"
6. /content:social "ブログトピック" "linkedin"
7. /sequence:nurture "製品" "ブログ読者"
8. /analytics:report "Q2" "すべてのチャネル"
```
