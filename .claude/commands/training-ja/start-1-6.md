## 言語と品質基準

**重要**: ユーザーが使用している言語で応答してください。日本語なら日本語で、スペイン語ならスペイン語で応答してください。

## Claudeへの指示

CLAUDE.mdと永続的なプロジェクトコンテキストの維持方法について学生に教えてください。

### レッスン概要

---

**モジュール1.6: プロジェクトメモリ**

CLAUDE.mdはClaudeに永続的なブリーフィングドキュメントを与えるようなものです。このプロジェクトで作業するたびに、Claudeはまずこのファイルを読み、それらのガイドラインを適用します。

**所要時間:** 約20分

---

### ステップ1: 現在のCLAUDE.mdを表示する

プロジェクトのCLAUDE.mdを読んでください：

```
このプロジェクトのCLAUDE.mdファイルを読む
```

各セクションを説明してください：
- 役割と責任
- ワークフロー（マーケティング、セールス、CRM）
- マーケティングエージェント
- スキルカタログ
- コマンドカテゴリ
- ドキュメント管理

### ステップ2: 仕組みを説明する

CLAUDE.mdが存在すると、Claudeは自動的に：
- どのエージェントが利用可能か知っている
- ワークフロー構造を理解している
- 適切なコマンドを参照する
- マーケティングルールに従う
- 正しいスキルを使用する

毎回Claudeに思い出させる必要はありません - 自動的です！

### ステップ3: 重要なCLAUDE.mdセクション

重要なセクションを説明してください：

**ワークフロー：**
```markdown
### コアワークフロー
- **マーケティング:** `./.claude/workflows/primary-workflow.md`
- **セールス:** `./.claude/workflows/sales-workflow.md`
- **CRM:** `./.claude/workflows/crm-workflow.md`
```

**エージェントマッピング：**
```markdown
### コアマーケティングエージェント
- `attraction-specialist` - TOFU（SEO、ランディングページ）
- `lead-qualifier` - インテント検出、スコアリング
- `email-wizard` - シーケンス、自動化
...
```

**コマンドカテゴリ：**
```markdown
### キャンペーン管理
- `/campaign:plan`、`/campaign:brief`、`/campaign:analyze`

### コンテンツ作成
- `/content:blog`、`/content:social`、`/content:email`
...
```

### ステップ4: コンテキスト認識をテストする

ブランドガイドラインに言及せずに質問してください：

```
リモートチームの生産性についての短いLinkedIn投稿を書いてください
```

出力が自動的に以下に一致することを指摘してください：
- ガイドラインからのブランドボイス
- ターゲットペルソナの言語
- 重要なメッセージングフレームワーク

### ステップ5: ワークフロー参照を理解する

ワークフローがどのように参照されるかを示してください：

```
.claude/workflows/primary-workflow.mdを読む
```

説明してください：
- マーケティングパイプラインステージ
- 各ステージでのエージェントの責任
- 品質ゲートとチェックポイント

### ステップ6: マーケティングルール

マーケティングルールを示してください：

```
.claude/workflows/marketing-rules.mdを読む
```

主要なルールを説明してください：
- トークン効率
- 多言語サポート
- 品質基準
- スキルアクティベーション

### ステップ7: プロジェクトコンテキストの利点

利点を要約してください：
- 自動的に一貫したブランドボイス
- 正しいエージェント選択
- 適切なコマンド使用
- ワークフローコンプライアンス
- 品質基準の施行

### ステップ8: メンテナンスのヒント

継続的なメンテナンスを説明してください：
- 新しいキャンペーンがローンチしたら更新
- 成功したコンテンツからの学びを追加
- 新しいドキュメントを参照
- エージェントリストを最新に保つ

## 主な教育ポイント
- CLAUDE.mdがClaudeに永続的なコンテキストを与えます
- ワークフロー、エージェント、コマンド、ルールが含まれます
- Claudeはすべての作業に自動的に適用します
- ワークフローがマーケティングプロセスを定義します
- マーケティングルールが品質基準を確保します
