## L'État d'Esprit du Marketeur 10x

La plupart des marketeurs travaillent de manière **linéaire** - chaque campagne nécessite le même effort.

Les **Marketeurs 10x** travaillent de manière **exponentielle** - chaque campagne rend la suivante plus rapide et meilleure.

Le secret : **La capitalisation systématique**

## Les Quatre Piliers

### Pilier 1 : Maîtrise des Commandes

Savoir quelles commandes enchaîner ensemble :

**Workflow de Lancement de Campagne :**
```
1. /research:market → Understand market
2. /research:persona → Define audience
3. /campaign:plan → Strategic plan
4. /campaign:brief → Creative direction
5. /campaign:calendar → Content schedule
6. /seo:keywords → SEO foundation
7. /content:blog → Pillar content
8. /content:social → Distribution content
9. /sequence:welcome → Lead nurture
10. /analytics:funnel → Measurement
```

**Workflow de Production de Contenu :**
```
1. /seo:keywords → Find opportunities
2. /content:blog → Create content
3. /seo:optimize → Optimize for search
4. /content:social → Repurpose for social
5. Review with agents → Quality check
```

### Pilier 2 : Orchestration des Agents

Utiliser les agents de manière stratégique :

**Pour la Phase de Recherche :**
- `researcher` → Compréhension du marché
- `attraction-specialist` → Intelligence compétitive

**Pour la Phase de Création :**
- `brainstormer` → Concepts de campagne
- `copywriter` → Création de contenu
- `planner` → Structure stratégique

**Pour la Phase de Révision :**
- `brand-voice-guardian` → Vérification de la voix
- `conversion-optimizer` → Révision CRO
- `seo-specialist` → Validation SEO
- Persona reviewers → Adéquation avec l'audience

**Pour la Phase d'Optimisation :**
- `lead-qualifier` → Qualité des leads
- `email-wizard` → Optimisation des séquences
- `continuity-specialist` → Rétention

### Pilier 3 : Effet de Levier de la Documentation

Utiliser la documentation pour capitaliser les connaissances :

**Avant chaque campagne :**
```
1. Read docs/campaign-playbooks.md
2. Read docs/content-style-guide.md
3. Read docs/channel-strategies.md
4. Review your pattern library
```

**Après chaque campagne :**
```
1. Update pattern library with learnings
2. Document winning approaches
3. Note what to avoid
4. Track performance trends
```

## Le Système d'Exécution 10x

### Quotidien : Victoires Rapides (15 min)
```
/ops:daily "focus area"
```
- Vérifier les métriques
- Créer 1 élément de contenu
- Mettre à jour la bibliothèque de patterns

### Hebdomadaire : Révision Stratégique (1 heure)
```
/ops:weekly "current week"
```
- Examiner les performances de campagne
- Planifier le contenu de la semaine suivante
- Mettre à jour les séquences en fonction des données

### Mensuel : Analyse Approfondie (2 heures)
```
/ops:monthly "current month"
/analytics:report "all campaigns" "full analysis"
```
- Revue complète des performances
- Mettre à jour la documentation
- Affiner les stratégies
