## Instructions pour Claude

Guidez les étudiants à travers de véritables tâches marketing : création de contenu multi-canal, analyse concurrentielle et planification de contenu en utilisant les commandes réelles du système.

### Aperçu de la Leçon

---

**Module 1.3 : Premières Tâches Marketing**

Passons maintenant à un vrai travail marketing. Vous allez accomplir trois tâches courantes que chaque marketeur effectue régulièrement.

**Durée :** ~30 minutes

---

### Tâche 1 : Génération de Contenu Multi-Canal

Générez du contenu pour plusieurs canaux en utilisant les commandes de contenu :

**Publication LinkedIn :**
```
/content:social "AgentKits launch announcement - team productivity coordination for remote teams" "linkedin"
```

**Article de Blog :**
```
/content:blog "How Remote Teams Can Coordinate Focus Time Without Endless Meetings" "remote team coordination"
```

**Email :**
```
/content:email "product announcement" "existing subscribers"
```

Examinez les résultats ensemble. Montrez l'itération :

```
Make the LinkedIn post more educational - focus on the problem of coordinating deep work time
```

### Tâche 2 : Analyse Concurrentielle

Utilisez la commande d'analyse concurrentielle :

```
/competitor:deep "RescueTime - personal time tracking and productivity software"
```

Expliquez ce que l'agent `researcher` analyse :
- Public cible
- Fonctionnalités clés et positionnement
- Modèle de tarification
- Forces et faiblesses
- Opportunités de marché

Posez une question de suivi :
```
Based on this analysis, what's AgentKits's biggest competitive advantage?
```

### Tâche 3 : Calendrier de Contenu

Utilisez la commande de calendrier de campagne :

```
/campaign:calendar "4 weeks - AgentKits product launch - focus on remote work productivity, team collaboration, deep work"
```

Examinez le calendrier généré :
- Sujets d'articles de blog avec mots-clés SEO
- Thèmes des réseaux sociaux par plateforme
- Planning des newsletters par email
- Objectifs de contenu pour chaque élément

### Étape 4 : Développer un Élément

Prenez un sujet et développez-le en utilisant les commandes de contenu :

```
/content:blog "The Complete Guide to Team Focus Time: How Remote Teams Can Coordinate Deep Work" "team focus time"
```

### Étape 5 : Optimisation SEO

Utilisez les commandes SEO pour optimiser :

```
/seo:keywords "remote team productivity"
```

Puis :
```
/seo:optimize "the blog post we just created" "team focus time"
```

### Étape 6 : Révision avec les Spécialistes

Utilisez les agents réviseurs (expliquez que ceux-ci seront couverts en détail plus tard) :

```
Review the blog post from three perspectives:
1. Brand Voice Guardian - does it match our voice?
2. SEO Specialist - is it optimized for search?
3. Conversion Optimizer - will it drive action?
```

### Célébration

Soulignez ce qu'ils viennent d'accomplir :
- Génération de contenu multi-canal en utilisant les commandes `/content:*`
- Analyse concurrentielle en utilisant `/competitor:deep`
- Calendrier de contenu en utilisant `/campaign:calendar`
- Recherche de mots-clés SEO en utilisant `/seo:keywords`
- Article de blog complet avec optimisation SEO

## Points Clés d'Enseignement
- Les commandes réelles gèrent de vraies tâches marketing
- Les commandes `/content:*` créent du contenu spécifique à chaque plateforme
- `/competitor:deep` fournit une intelligence concurrentielle
- `/campaign:calendar` crée des calendriers de contenu
- Les commandes `/seo:*` gèrent l'optimisation pour les moteurs de recherche
- Fournissez toujours le contexte (marque, audience, objectifs)
