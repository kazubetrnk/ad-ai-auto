### Aperçu de la Leçon

---

**Module 2.6 : Optimisation SEO**

Le SEO génère une croissance durable à long terme. Maîtrisons la recherche de mots-clés, l'optimisation de contenu et le SEO technique.

**Durée :** ~40 minutes

---

### Étape 1 : Recherche de Mots-Clés

Utilisez la commande de recherche de mots-clés :

```
/seo:keywords "remote team productivity"
```

```
/seo:keywords "team focus time software"
```

```
/seo:keywords "coordination tools for remote teams"
```

Examinez les opportunités de mots-clés :
- Estimations du volume de recherche
- Niveaux de concurrence
- Classification de l'intention
- Groupes de mots-clés

### Étape 2 : Analyse SEO des Concurrents

Analysez le SEO des concurrents :

```
/seo:competitor "rescuetime.com"
```

```
/seo:competitor "freedom.to"
```

Identifiez :
- Les mots-clés pour lesquels ils se positionnent
- Les lacunes de contenu
- Les opportunités de backlinks
- Les forces/faiblesses techniques

### Étape 3 : Audit SEO

Exécutez un audit complet :

```
/seo:audit "agentkits.com"
```

Examinez les domaines de l'audit :
- SEO technique (capacité d'exploration, vitesse, mobile)
- SEO on-page (titres, méta, en-têtes)
- Qualité du contenu (profondeur, pertinence)
- Profil de backlinks

### Étape 4 : Optimisation du Contenu

Optimisez le contenu existant :

```
/seo:optimize "blog post about team productivity" "remote team productivity"
```

L'optimisation comprend :
- Amélioration de la balise titre
- Méta-description
- Structure des en-têtes (H1, H2, H3)
- Placement des mots-clés
- Suggestions de liens internes
- Texte alternatif des images

### Étape 5 : Créer du Contenu SEO

Utilisez la commande blog avec un focus SEO :

```
/content:blog "The Complete Guide to Remote Team Productivity" "remote team productivity"
```

Puis optimisez :

```
/seo:optimize "the blog post" "remote team productivity"
```

### Étape 6 : Stratégie de Featured Snippet

Optimisez pour les featured snippets :

```
Create content optimized for featured snippets:

Target query: "How to improve remote team productivity"

PARAGRAPH SNIPPET (40-60 words):
Direct answer format

LIST SNIPPET (numbered steps):
Step-by-step format

TABLE SNIPPET:
Comparison format
```

### Étape 7 : Liste de Vérification SEO Technique

Utilisez la liste de vérification SEO :

```
/checklist:seo-weekly "agentkits.com"
```

Tâches SEO hebdomadaires :
- Vérification des erreurs d'exploration
- Surveillance de la vitesse de page
- Utilisabilité mobile
- Statut d'indexation
- Suivi des mots-clés

### Étape 8 : Stratégie Pilier-Cluster

Créez un plan de contenu pilier :

```
Create pillar-cluster SEO strategy:

PILLAR PAGE: "The Complete Guide to Remote Team Productivity"
- 3000-5000 words, comprehensive

CLUSTER CONTENT (10 posts):
1. "How to Schedule Team Focus Time" → team focus time
2. "Reducing Meetings Without Losing Alignment" → reduce meetings remote
3. "Async Communication Best Practices" → async communication
4. "Deep Work for Remote Teams" → deep work teams
5. "Team Productivity Metrics That Matter" → productivity metrics
...

INTERNAL LINKING:
- Each cluster links to pillar
- Pillar links to all clusters
- Cross-links between related clusters
```

### Étape 9 : Calendrier de Contenu avec SEO

Créez un calendrier axé sur le SEO :

```
/campaign:calendar "12 weeks - SEO content strategy - target keywords: remote team productivity, team focus time, coordination software"
```

Associez le contenu aux mots-clés :
- Semaine 1 : [Mot-clé] - [Titre] - [Type]
- Semaine 2 : [Mot-clé] - [Titre] - [Type]
...

### Étape 10 : Mesure SEO

Suivez les performances SEO :

```
/analytics:report "SEO performance" "organic traffic, keyword rankings, conversions"
```

Métriques clés :
- Tendance du trafic organique
- Changements de position des mots-clés
- Taux de conversion organique
- Performance des pages
