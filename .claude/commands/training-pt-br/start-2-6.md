### Visão Geral da Lição

---

**Módulo 2.6: Otimização de SEO**

SEO impulsiona crescimento sustentável de longo prazo. Vamos dominar pesquisa de palavras-chave, otimização de conteúdo e SEO técnico.

**Duração:** ~40 minutos

---

### Passo 1: Pesquisa de Palavras-chave

Use o comando de pesquisa de palavras-chave:

```
/seo:keywords "remote team productivity"
```

```
/seo:keywords "team focus time software"
```

```
/seo:keywords "coordination tools for remote teams"
```

Revise as oportunidades de palavras-chave:
- Estimativas de volume de busca
- Níveis de concorrência
- Classificação de intenção
- Clusters de palavras-chave

### Passo 2: Análise de SEO de Concorrentes

Analise o SEO dos concorrentes:

```
/seo:competitor "rescuetime.com"
```

```
/seo:competitor "freedom.to"
```

Identifique:
- Palavras-chave para as quais eles ranqueiam
- Lacunas de conteúdo
- Oportunidades de backlinks
- Pontos fortes/fracos técnicos

### Passo 3: Auditoria de SEO

Execute uma auditoria abrangente:

```
/seo:audit "agentkits.com"
```

Revise as áreas da auditoria:
- SEO técnico (rastreabilidade, velocidade, mobile)
- SEO on-page (títulos, meta, cabeçalhos)
- Qualidade do conteúdo (profundidade, relevância)
- Perfil de backlinks

### Passo 4: Otimização de Conteúdo

Otimize conteúdo existente:

```
/seo:optimize "blog post about team productivity" "remote team productivity"
```

A otimização inclui:
- Melhoria da tag de título
- Meta descrição
- Estrutura de cabeçalhos (H1, H2, H3)
- Posicionamento de palavras-chave
- Sugestões de links internos
- Texto alternativo de imagens

### Passo 5: Criar Conteúdo SEO

Use o comando de blog com foco em SEO:

```
/content:blog "The Complete Guide to Remote Team Productivity" "remote team productivity"
```

Depois otimize:

```
/seo:optimize "the blog post" "remote team productivity"
```

### Passo 6: Estratégia de Featured Snippet

Otimize para featured snippets:

```
Create content optimized for featured snippets:

Target query: "How to improve remote team productivity"

PARAGRAPH SNIPPET (40-60 words):
Direct answer format

LIST SNIPPET (numbered steps):
Step-by-step format

TABLE SNIPPET:
Comparison format
```

### Passo 7: Checklist de SEO Técnico

Use o checklist de SEO:

```
/checklist:seo-weekly "agentkits.com"
```

Tarefas semanais de SEO:
- Verificação de erros de rastreamento
- Monitoramento de velocidade da página
- Usabilidade mobile
- Status de indexação
- Rastreamento de palavras-chave

### Passo 8: Estratégia Pilar-Cluster

Crie um plano de conteúdo pilar:

```
Create pillar-cluster SEO strategy:

PILLAR PAGE: "The Complete Guide to Remote Team Productivity"
- 3000-5000 words, comprehensive

CLUSTER CONTENT (10 posts):
1. "How to Schedule Team Focus Time" → team focus time
2. "Reducing Meetings Without Losing Alignment" → reduce meetings remote
3. "Async Communication Best Practices" → async communication
4. "Deep Work for Remote Teams" → deep work teams
5. "Team Productivity Metrics That Matter" → productivity metrics
...

INTERNAL LINKING:
- Each cluster links to pillar
- Pillar links to all clusters
- Cross-links between related clusters
```

### Passo 9: Calendário de Conteúdo com SEO

Crie um calendário focado em SEO:

```
/campaign:calendar "12 weeks - SEO content strategy - target keywords: remote team productivity, team focus time, coordination software"
```

Mapeie conteúdo para palavras-chave:
- Semana 1: [Palavra-chave] - [Título] - [Tipo]
- Semana 2: [Palavra-chave] - [Título] - [Tipo]
...

### Passo 10: Medição de SEO

Acompanhe o desempenho de SEO:

```
/analytics:report "SEO performance" "organic traffic, keyword rankings, conversions"
```

Métricas-chave:
- Tendência de tráfego orgânico
- Mudanças de posição de palavras-chave
- Taxa de conversão orgânica
- Desempenho de páginas
