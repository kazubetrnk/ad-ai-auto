**Módulo 1.7: Navegação e Busca**

### Passo 1: Encontrando Comandos

Pratique a descoberta de comandos:

```
What commands are available for campaigns?
```

```
Show me all SEO-related commands
```

```
What commands help with email marketing?
```

### Passo 2: Referência de Categorias de Comandos

Revise todas as categorias de comandos:

| Categoria | Comandos | Casos de Uso |
|----------|----------|-----------|
| `/campaign:*` | plan, brief, analyze, calendar | Gerenciamento de campanhas |
| `/content:*` | blog, social, email, landing, ads | Criação de conteúdo |
| `/seo:*` | keywords, audit, optimize, competitor | Otimização de SEO |
| `/analytics:*` | roi, funnel, report | Rastreamento de desempenho |
| `/sales:*` | pitch, outreach, battlecard, qualify | Habilitação de vendas |
| `/leads:*` | score, nurture, qualify | Gerenciamento de leads |
| `/crm:*` | sequence, segment, score, lifecycle | Automação de CRM |

| `/brand:*` | voice, book, assets | Gerenciamento de marca |
| `/ops:*` | daily, weekly, monthly | Operações |
| `/report:*` | weekly, monthly | Relatórios |

```

```

### Passo 5: Comandos de Referência Rápida

Compartilhe atalhos de comandos essenciais:

**Conteúdo Rápido:**
```
/content:fast "quick headline for social post"
```

**Conteúdo de Qualidade:**
```
/content:good "detailed blog post with research"
```

**Brainstorming:**
```
/brainstorm "marketing channel ideas for B2B SaaS"
```

### Passo 6: Comandos Operacionais

Pratique fluxos de trabalho operacionais:

**Operações Diárias:**
```
/ops:daily "content creation"
```

**Revisão Semanal:**
```
/ops:weekly "current week dates"
```

**Checklists de Campanha:**
```
/checklist:campaign-launch "Q1 Product Launch" "next week"
```

### Passo 7: Fluxo de Trabalho Completo de Comandos

Demonstre um fluxo de trabalho completo:

```
1. /research:market "B2B productivity software"
2. /campaign:plan "Q2 Growth Campaign"
3. /seo:keywords "team productivity"
4. /content:blog "productivity topic" "target keyword"
5. /seo:optimize "the blog post" "keyword"
6. /content:social "blog topic" "linkedin"
7. /sequence:nurture "product" "blog readers"
8. /analytics:report "Q2" "all channels"
```
