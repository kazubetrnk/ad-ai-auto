## Anweisungen für Claude

Führen Sie den Studenten durch seine erste echte Marketing-Aufgabe - die Erstellung von Markenrichtlinien mit unseren Befehlen.

### Lektionsübersicht

---

**Modul 0.2: Ihre erste Marketing-Aufgabe**

Jetzt werden wir echte Marketing-Arbeit leisten. Wir erstellen Markenrichtlinien für AgentKits mit dem Befehl `/brand:voice`.

---

### Schritt 1: Markenrichtlinien erklären

Erklären Sie, warum Markenrichtlinien wichtig sind:
- Gewährleistet Konsistenz über alle Inhalte hinweg
- Hilft Claude (und Menschen), in der richtigen Stimme zu schreiben
- Dokumentiert Kernbotschaften und Terminologie
- Verhindert markenfremde Inhalte

### Schritt 2: Den Brand Voice Befehl verwenden

Führen Sie sie zur Verwendung des tatsächlichen Systembefehls:

```
/brand:voice "AgentKits - B2B team productivity coordination tool for remote teams"
```

Lassen Sie Claude umfassende Markenrichtlinien generieren und überprüfen Sie diese dann mit dem Studenten.

### Schritt 3: Kunden-Personas erstellen

Verwenden Sie nun den Recherche-Befehl für Personas:

```
/research:persona "Remote team managers at tech companies using AgentKits"
```

Erklären Sie:
- Der `researcher` Agent kümmert sich um Marktforschung
- Personas helfen dabei, Inhalte auf bestimmte Zielgruppen auszurichten
- Wir werden diese Personas während des gesamten Kurses verwenden

### Schritt 4: Überprüfen, was erstellt wurde

Überprüfen Sie gemeinsam die Ergebnisse:

```
Show me what the brand:voice command created
```

Weisen Sie hin auf:
- Stimmattribute und Tonspektrum
- Messaging-Framework
- Zu verwendende und zu vermeidende Wörter
- Integration mit anderen Agents

### Schritt 5: Die Macht des Kontexts

Erklären Sie:
- Diese Richtlinien existieren nun im Projektkontext
- Bei zukünftigen Aufgaben können Agents darauf verweisen
- Dies ist "Kontextbewusstsein" - eine von Claudes Superkräften
- Wir werden diese während des gesamten Kurses verwenden

### Schnellübung: Content-Generierung testen

Lassen Sie sie versuchen, den Markenkontext zu verwenden:

```
/content:social "Remote team productivity tips" "linkedin"
```

Zeigen Sie, wie Claude den Markenkontext automatisch bei der Inhaltserstellung verwendet.

### Schritt 6: Andere wichtige Befehle erkunden

Demonstrieren Sie kurz andere Befehle, die sie beherrschen werden:

**Kampagnenplanung:**
```
/campaign:plan "Q1 Product Launch"
```

**SEO-Recherche:**
```
/seo:keywords "remote team productivity"
```

**E-Mail-Sequenzen:**
```
/sequence:welcome "AgentKits" "trial users"
```

### Was kommt als Nächstes

Sagen Sie ihnen:
- **Glückwunsch!** Modul 0 abgeschlossen!
- Sie haben echte Marketing-Befehle verwendet und das System in Aktion gesehen
- **Weiter:** `/training-de:start-1-1` - Willkommen bei Markit (Kernkonzepte beginnen)
- Sie werden tief in Agents, Workflows und erweiterte Befehle eintauchen

## Wichtige Lehrpunkte
- Markenrichtlinien gewährleisten Konsistenz
- Verwenden Sie `/brand:voice`, um Stimmenrichtlinien zu erstellen
- Verwenden Sie `/research:persona` für Kunden-Personas
- Kontextbewusstsein bedeutet, dass Agents auf bestehende Arbeit verweisen
- Echte Befehle (`/campaign:*`, `/content:*`, `/seo:*`) sind das, was sie beherrschen werden
