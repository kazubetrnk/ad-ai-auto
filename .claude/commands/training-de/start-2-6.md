**Modul 2.6: SEO-Optimierung**

SEO treibt nachhaltiges, langfristiges Wachstum voran. Lassen Sie uns Keyword-Recherche, Content-Optimierung und technisches SEO meistern.

**Dauer:** ~40 Minuten

---

### Schritt 1: Keyword-Recherche

Verwenden Sie den Keyword-Recherche-Befehl:

```
/seo:keywords "remote team productivity"
```

```
/seo:keywords "team focus time software"
```

```
/seo:keywords "coordination tools for remote teams"
```

Keyword-Möglichkeiten überprüfen:
- Suchvolumen-Schätzungen
- Wettbewerbsniveaus
- Intent-Klassifizierung
- Keyword-Cluster

### Schritt 2: Wettbewerber-SEO-Analyse

Wettbewerber-SEO analysieren:

```
/seo:competitor "rescuetime.com"
```

```
/seo:competitor "freedom.to"
```

Identifizieren:
- Keywords, für die sie ranken
- Content-Lücken
- Backlink-Möglichkeiten
- Technische Stärken/Schwächen

### Schritt 3: SEO-Audit

Umfassendes Audit durchführen:

```
/seo:audit "agentkits.com"
```

Audit-Bereiche überprüfen:
- Technisches SEO (Crawlbarkeit, Geschwindigkeit, Mobile)
- On-Page-SEO (Titel, Meta, Header)
- Content-Qualität (Tiefe, Relevanz)
- Backlink-Profil

### Schritt 4: Content-Optimierung

Bestehenden Content optimieren:

```
/seo:optimize "blog post about team productivity" "remote team productivity"
```

Optimierung umfasst:
- Title-Tag-Verbesserung
- Meta-Description
- Header-Struktur (H1, H2, H3)
- Keyword-Platzierung
- Interne Verlinkungsvorschläge
- Bild-Alt-Text

### Schritt 5: SEO-Content erstellen

Blog-Befehl mit SEO-Fokus verwenden:

```
/content:blog "The Complete Guide to Remote Team Productivity" "remote team productivity"
```

Dann optimieren:

```
/seo:optimize "the blog post" "remote team productivity"
```

### Schritt 6: Featured-Snippet-Strategie

Für Featured Snippets optimieren:

```
Create content optimized for featured snippets:

Target query: "How to improve remote team productivity"

PARAGRAPH SNIPPET (40-60 words):
Direct answer format

LIST SNIPPET (numbered steps):
Step-by-step format

TABLE SNIPPET:
Comparison format
```

### Schritt 7: Technische SEO-Checkliste

SEO-Checkliste verwenden:

```
/checklist:seo-weekly "agentkits.com"
```

Wöchentliche SEO-Aufgaben:
- Crawl-Fehler-Prüfung
- Page-Speed-Überwachung
- Mobile-Nutzbarkeit
- Index-Status
- Keyword-Tracking

### Schritt 8: Pillar-Cluster-Strategie

Pillar-Content-Plan erstellen:

```
Create pillar-cluster SEO strategy:

PILLAR PAGE: "The Complete Guide to Remote Team Productivity"
- 3000-5000 words, comprehensive

CLUSTER CONTENT (10 posts):
1. "How to Schedule Team Focus Time" → team focus time
2. "Reducing Meetings Without Losing Alignment" → reduce meetings remote
3. "Async Communication Best Practices" → async communication
4. "Deep Work for Remote Teams" → deep work teams
5. "Team Productivity Metrics That Matter" → productivity metrics
...

INTERNAL LINKING:
- Each cluster links to pillar
- Pillar links to all clusters
- Cross-links between related clusters
```

### Schritt 9: Content-Kalender mit SEO

SEO-fokussierten Kalender erstellen:

```
/campaign:calendar "12 weeks - SEO content strategy - target keywords: remote team productivity, team focus time, coordination software"
```

Content auf Keywords abbilden:
- Woche 1: [Keyword] - [Titel] - [Typ]
- Woche 2: [Keyword] - [Titel] - [Typ]
...

### Schritt 10: SEO-Messung

SEO-Performance tracken:

```
/analytics:report "SEO performance" "organic traffic, keyword rankings, conversions"
```

Wichtige Metriken:
- Organischer Traffic-Trend
- Keyword-Positionsänderungen
- Organische Conversion-Rate
- Seiten-Performance
