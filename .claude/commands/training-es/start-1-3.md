## Instrucciones para Claude

Guía a los estudiantes a través de tareas reales de marketing: redacción multicanal, análisis competitivo y planificación de contenido usando comandos reales del sistema.

### Descripción General de la Lección

---

**Módulo 1.3: Primeras Tareas de Marketing**

Ahora hagamos trabajo real de marketing. Completarás tres tareas comunes que todo especialista en marketing hace regularmente.

**Duración:** ~30 minutos

---

### Tarea 1: Generación de Redacción Multicanal

Genera redacción para múltiples canales usando comandos de contenido:

**Publicación de LinkedIn:**
```
/content:social "AgentKits launch announcement - team productivity coordination for remote teams" "linkedin"
```

**Artículo de Blog:**
```
/content:blog "How Remote Teams Can Coordinate Focus Time Without Endless Meetings" "remote team coordination"
```

**Correo Electrónico:**
```
/content:email "product announcement" "existing subscribers"
```

Revisa los resultados juntos. Muestra la iteración:

```
Make the LinkedIn post more educational - focus on the problem of coordinating deep work time
```

### Tarea 2: Análisis Competitivo

Usa el comando de análisis competitivo:

```
/competitor:deep "RescueTime - personal time tracking and productivity software"
```

Explica qué analiza el agente `researcher`:
- Audiencia objetivo
- Características clave y posicionamiento
- Modelo de precios
- Fortalezas y debilidades
- Oportunidades de mercado

Haz una pregunta de seguimiento:
```
Based on this analysis, what's AgentKits's biggest competitive advantage?
```

### Tarea 3: Calendario de Contenido

Usa el comando de calendario de campaña:

```
/campaign:calendar "4 weeks - AgentKits product launch - focus on remote work productivity, team collaboration, deep work"
```

Revisa el calendario generado:
- Temas de artículos de blog con palabras clave SEO
- Temas de redes sociales por plataforma
- Calendario de boletines por correo electrónico
- Objetivos de contenido para cada pieza

### Paso 4: Expandir una Pieza

Toma un tema y expándelo usando comandos de contenido:

```
/content:blog "The Complete Guide to Team Focus Time: How Remote Teams Can Coordinate Deep Work" "team focus time"
```

### Paso 5: Optimización SEO

Usa comandos SEO para optimizar:

```
/seo:keywords "remote team productivity"
```

Luego:
```
/seo:optimize "the blog post we just created" "team focus time"
```

### Paso 6: Revisar con Especialistas

Usa agentes revisores (explica que estos se cubrirán en detalle más adelante):

```
Review the blog post from three perspectives:
1. Brand Voice Guardian - does it match our voice?
2. SEO Specialist - is it optimized for search?
3. Conversion Optimizer - will it drive action?
```

### Celebra

Señala lo que acaban de lograr:
- Generación de redacción multicanal usando comandos `/content:*`
- Análisis competitivo usando `/competitor:deep`
- Calendario de contenido usando `/campaign:calendar`
- Investigación de palabras clave SEO usando `/seo:keywords`
- Artículo de blog completo con optimización SEO

## Puntos Clave de Enseñanza
- Los comandos reales manejan tareas reales de marketing
- Los comandos `/content:*` crean contenido específico para cada plataforma
- `/competitor:deep` proporciona inteligencia competitiva
- `/campaign:calendar` crea calendarios de contenido
- Los comandos `/seo:*` manejan la optimización para motores de búsqueda
- Siempre proporciona contexto (marca, audiencia, objetivos)
