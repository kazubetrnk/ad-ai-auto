## La Mentalidad del Marketer 10x

La mayoría de los marketers trabajan de forma **lineal** - cada campaña requiere el mismo esfuerzo.

Los **Marketers 10x** trabajan de forma **exponencial** - cada campaña hace que la siguiente sea más rápida y mejor.

El secreto: **Composición sistemática**

## Los Cuatro Pilares

### Pilar 1: Dominio de Comandos

Saber qué comandos encadenar juntos:

**Flujo de Lanzamiento de Campaña:**
```
1. /research:market → Understand market
2. /research:persona → Define audience
3. /campaign:plan → Strategic plan
4. /campaign:brief → Creative direction
5. /campaign:calendar → Content schedule
6. /seo:keywords → SEO foundation
7. /content:blog → Pillar content
8. /content:social → Distribution content
9. /sequence:welcome → Lead nurture
10. /analytics:funnel → Measurement
```

**Flujo de Producción de Contenido:**
```
1. /seo:keywords → Find opportunities
2. /content:blog → Create content
3. /seo:optimize → Optimize for search
4. /content:social → Repurpose for social
5. Review with agents → Quality check
```

### Pilar 2: Orquestación de Agentes

Usa los agentes estratégicamente:

**Para la Fase de Investigación:**
- `researcher` → Comprensión del mercado
- `attraction-specialist` → Inteligencia competitiva

**Para la Fase de Creación:**
- `brainstormer` → Conceptos de campaña
- `copywriter` → Creación de contenido
- `planner` → Estructura estratégica

**Para la Fase de Revisión:**
- `brand-voice-guardian` → Verificación de voz
- `conversion-optimizer` → Revisión CRO
- `seo-specialist` → Validación SEO
- Revisores de persona → Ajuste de audiencia

**Para la Fase de Optimización:**
- `lead-qualifier` → Calidad de leads
- `email-wizard` → Optimización de secuencias
- `continuity-specialist` → Retención

### Pilar 3: Apalancamiento de Documentación

Usa la documentación para componer conocimiento:

**Antes de cada campaña:**
```
1. Read docs/campaign-playbooks.md
2. Read docs/content-style-guide.md
3. Read docs/channel-strategies.md
4. Review your pattern library
```

**Después de cada campaña:**
```
1. Update pattern library with learnings
2. Document winning approaches
3. Note what to avoid
4. Track performance trends
```

## El Sistema de Ejecución 10x

### Diario: Victorias Rápidas (15 min)
```
/ops:daily "focus area"
```
- Revisar métricas
- Crear 1 pieza de contenido
- Actualizar biblioteca de patrones

### Semanal: Revisión Estratégica (1 hora)
```
/ops:weekly "current week"
```
- Revisar rendimiento de campañas
- Planificar contenido de la próxima semana
- Actualizar secuencias basadas en datos

### Mensual: Análisis Profundo (2 horas)
```
/ops:monthly "current month"
/analytics:report "all campaigns" "full analysis"
```
- Revisión integral de rendimiento
- Actualizar documentación
- Refinar estrategias
