### Resumen de la Lección

---

**Módulo 1.7: Navegación y Búsqueda**

La navegación eficiente es crucial a medida que tus proyectos de marketing crecen. Dominemos la búsqueda de comandos y la gestión de flujos de trabajo.

**Duración:** ~20 minutos

---

### Paso 1: Encontrar Comandos

Practica el descubrimiento de comandos:

```
What commands are available for campaigns?
```

```
Show me all SEO-related commands
```

```
What commands help with email marketing?
```

### Paso 2: Referencia de Categorías de Comandos

Revisa todas las categorías de comandos:

| Categoría | Comandos | Casos de Uso |
|----------|----------|-----------|
| `/campaign:*` | plan, brief, analyze, calendar | Gestión de campañas |
| `/content:*` | blog, social, email, landing, ads | Creación de contenido |
| `/seo:*` | keywords, audit, optimize, competitor | Optimización SEO |
| `/analytics:*` | roi, funnel, report | Seguimiento de rendimiento |
| `/sales:*` | pitch, outreach, battlecard, qualify | Habilitación de ventas |
| `/leads:*` | score, nurture, qualify | Gestión de leads |
| `/crm:*` | sequence, segment, score, lifecycle | Automatización de CRM |
| `/sequence:*` | welcome, nurture, re-engage | Secuencias de email |
| `/social:*` | engage, viral, schedule | Redes sociales |
| `/research:*` | market, persona, trend | Investigación de mercado |
| `/brand:*` | voice, book, assets | Gestión de marca |
| `/ops:*` | daily, weekly, monthly | Operaciones |
| `/report:*` | weekly, monthly | Reportes |
| `/checklist:*` | campaign-launch, seo-weekly | Listas de verificación |

### Paso 4: Encontrar Documentación

Practica encontrar documentación relevante:

```

```

```

```

```
Show me the content style guide
```

### Paso 5: Comandos de Referencia Rápida

Comparte atajos de comandos esenciales:

**Contenido Rápido:**
```
/content:fast "quick headline for social post"
```

**Contenido de Calidad:**
```
/content:good "detailed blog post with research"
```

**Lluvia de Ideas:**
```
/brainstorm "marketing channel ideas for B2B SaaS"
```

### Paso 6: Comandos Operacionales

Practica flujos de trabajo operacionales:

**Operaciones Diarias:**
```
/ops:daily "content creation"
```

**Revisión Semanal:**
```
/ops:weekly "current week dates"
```

**Listas de Verificación de Campañas:**
```
/checklist:campaign-launch "Q1 Product Launch" "next week"
```

### Paso 7: Flujo de Trabajo Completo de Comandos

Demuestra un flujo de trabajo completo:

```
1. /research:market "B2B productivity software"
2. /campaign:plan "Q2 Growth Campaign"
3. /seo:keywords "team productivity"
4. /content:blog "productivity topic" "target keyword"
5. /seo:optimize "the blog post" "keyword"
6. /content:social "blog topic" "linkedin"
7. /sequence:nurture "product" "blog readers"
8. /analytics:report "Q2" "all channels"
```
