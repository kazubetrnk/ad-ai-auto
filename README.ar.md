> *مستوحاة من حركة "البرمجة بالأجواء" (Vibe Coding) من المطورين... نحن نوسع الكون: **التسويق بالأجواء** (Vibe Marketing) لعصر الذكاء الاصطناعي حيث يعمل كل شيء بسلاسة.*

| | |
|---|---|
| **مع الذكاء الاصطناعي** | دع وكلاء الذكاء الاصطناعي يتولون حملاتك بينما تركز أنت على الاستراتيجية. فقط استرخِ ودع الوكلاء يقومون بالعمل الشاق. |
| **بدون الذكاء الاصطناعي** | هذا المستودع هو **مكتبة مرجعية شاملة** لأفضل الممارسات التسويقية والأطر والقوالب. استخدم وثائق المهارات كدليل تسويقي لك. |

## ما بداخل المستودع

يعمل مع **Claude Code** و **Cursor** و **GitHub Copilot** وأي مساعد ذكاء اصطناعي يدعم الوكلاء والمهارات. قم بالتثبيت كإضافة أو انسخ المكونات يدويًا.

```

|-- .claude-plugin/      # ملفات البيانات الوصفية للإضافة والسوق
|   |-- plugin.json            # بيانات الإضافة ومسارات المكونات
|   |-- marketplace.json       # كتالوج السوق لـ /plugin marketplace add
|
|-- .claude/
|   |-- agents/          # 18 وكيل تسويقي متخصص
|   |   |-- attraction-specialist.md    # توليد العملاء المحتملين، تحسين محركات البحث، صفحات الهبوط
|   |   |-- lead-qualifier.md           # تقييم وتصنيف العملاء المحتملين
|   |   |-- email-wizard.md             # تسلسل البريد الإلكتروني، الأتمتة
|   |   |-- sales-enabler.md            # ضمانات المبيعات، بطاقات المعارك
|   |   |-- continuity-specialist.md    # الاحتفاظ، إعادة المشاركة
|   |   |-- upsell-maximizer.md         # توسيع الإيرادات
|   |   |-- copywriter.md               # نصوص عالية التحويل
|   |   |-- conversion-optimizer.md     # متخصص تحسين التحويل
|   |   |-- seo-specialist.md           # تحسين محركات البحث
|   |   |-- brand-voice-guardian.md     # الاتساق في هوية العلامة التجارية
|   |   |-- ...والمزيد
|   |
|   |-- commands/        # 93 أمر مصنف حسب الفئة
|   |   |-- campaign/    # /campaign:plan, /campaign:brief, /campaign:analyze
|   |   |-- content/     # /content:blog, /content:landing, /content:email
|   |   |-- seo/         # /seo:keywords, /seo:audit, /seo:programmatic
|   |   |-- cro/         # /cro:page, /cro:form, /cro:popup, /cro:signup
```

## التثبيت

### الخيار 1: التثبيت عبر npx (موصى به)

أمر واحد لتثبيت 18 وكيل، 28 مهارة، 93 أمر:

```bash
npx @aitytech/agentkits-marketing install
```

**التثبيت الخاص بمنصة معينة:**

```bash
npx @aitytech/agentkits-marketing install --platform claude    # Claude Code
npx @aitytech/agentkits-marketing install --platform cursor    # Cursor IDE
npx @aitytech/agentkits-marketing install --platform windsurf  # Windsurf
npx @aitytech/agentkits-marketing install --platform cline     # Cline
npx @aitytech/agentkits-marketing install --platform copilot   # GitHub Copilot
npx @aitytech/agentkits-marketing install --platform all       # جميع المنصات
```

**أوامر CLI أخرى:**

```bash
npx @aitytech/agentkits-marketing --help        # إظهار جميع الأوامر
npx @aitytech/agentkits-marketing list-ides     # سرد بيئات التطوير المدعومة
```

## التدريب

**22 درسًا تفاعليًا** لإتقان التسويق المدعوم بالذكاء الاصطناعي. تعلم من خلال القيام بأعمال تسويقية حقيقية داخل مساعد الذكاء الاصطناعي الخاص بك.

| | |
|---|---|
| **الطريقة** | دروس تفاعلية يدرسها Claude |
| **المشروع** | وكالة Markit تعمل لصالح عميل AgentKits |
| **المدة** | 5-6 ساعات إجمالاً |
| **المتطلبات الأساسية** | Claude Code أو Cursor أو مساعد ذكاء اصطناعي متوافق |
| **اللغات** | الإنجليزية، الفيتنامية (Tiếng Việt)، اليابانية (日本語) |

```bash
# ابدأ التدريب الآن
/training:start-0-0           # الإنجليزية
/training-vi:start-0-0        # الفيتنامية (Tiếng Việt)
/training-ja:start-0-0        # اليابانية (日本語)
```

### مشروع التدريب: وكالة Markit

أنت استراتيجي تسويق في **Markit**، وكالة تسويق SaaS للشركات.

**عميلك:** AgentKits - مجموعة أدوات أتمتة التسويق بالذكاء الاصطناعي

| | |
|---|---|
| **المنتج** | أتمتة تسويقية ذكية على مستوى المؤسسات |
| **الهدف** | مؤسسو SaaS والمسوقون وفرق النمو |
| **التسعير** | مجاني ومفتوح المصدر (ترخيص MIT) |
| **المنافسون** | Jasper، Copy.ai، HubSpot |

**الشخصيات الرئيسية:**
- **Solo Sam** (25-35) - رائد أعمال منفرد/هاكر مستقل، SaaS ممول ذاتيًا
- **Marketer Maya** (30-40) - مدير تسويق، شركة SaaS متوسطة الحجم
- **Founder Felix** (28-40) - مؤسس تقني، شركة ناشئة في مرحلة مبكرة

- مكتبة محتوى كاملة (مدونة، بريد إلكتروني، وسائل التواصل الاجتماعي، إعلانات)
- تقرير تحليل تنافسي
- خطة تحسين محركات البحث
- لوحة تحكم تحليلات الحملة

---

### الوحدة 3: تحسين التحويل (60 دقيقة)

إتقان تحسين معدل التحويل بمهارات CRO المتخصصة.

| الأمر | الدرس | المدة |
|---------|--------|----------|
| `/training:start-3-1` | أساسيات تحسين التحويل | 20 دقيقة |
| `/training:start-3-2` | تحسين النماذج والتسجيل | 20 دقيقة |
| `/training:start-3-3` | تحسين النوافذ المنبثقة والإعداد | 20 دقيقة |

**ما ستتعلمه:**
- 7 مهارات CRO لقمع التحويل الكامل
- تحسين النماذج (قاعدة الحقول الخمسة)
- أفضل ممارسات تدفق التسجيل
- تصميم النوافذ المنبثقة والمحفزات
- الإعداد والتفعيل
- شاشات الحوائط الدفعة والترقية
- تصميم اختبار A/B

**ما ستبنيه:**
- تدقيق CRO لصفحة الهبوط
- تصميم نموذج محسّن
- تدفق الإعداد
- شاشة الترقية
- فرضيات اختبار A/B

**تغطية قمع CRO الكامل:**
```
الزائر → تحسين الصفحة → تحسين النموذج → تحسين التسجيل
     ↓
  تحسين النوافذ المنبثقة (جذب المنسحبين)
     ↓
مستخدم جديد → تحسين الإعداد → التفعيل
     ↓
مستخدم مجاني → تحسين الحائط الدفعة → عميل مدفوع
```

### التدريب متعدد اللغات

التدريب متاح بـ 3 لغات. جميع المحتويات متطابقة - اختر لغتك المفضلة:

| اللغة | بادئة الأمر | مثال |
|----------|---------------|---------|
| **الإنجليزية** | `/training:` | `/training:start-0-0` |
| **الفيتنامية** (Tiếng Việt) | `/training-vi:` | `/training-vi:start-0-0` |
| **اليابانية** (日本語) | `/training-ja:` | `/training-ja:start-0-0` |

**الأوامر المترجمة المتاحة:**
- `start-0-0` إلى `start-0-2` (الوحدة 0)
- `start-1-1` إلى `start-1-7` (الوحدة 1)
- `start-2-1` إلى `start-2-6` (الوحدة 2)
- `start-3-1` إلى `start-3-3` (الوحدة 3)
- `help`, `bonus-patterns`, `bonus-secret`, `persona-builder`

| **إطلاق منتج** | `/growth:launch`, `/campaign:plan` |
| **تحسين التسعير** | `/pricing:strategy` |
| **توسيع نطاق تحسين محركات البحث** | `/seo:programmatic`, `/seo:schema` |
| **تصميم الإحالات** | `/growth:referral` |
| **اختبار A/B** | `/test:ab-setup` |
