## 1. Brand Foundation

### Brand Positioning

> **For** SaaS founders, marketers, and growth teams **who** struggle to create consistent, high-quality marketing content at scale, **AgentKits** is an **AI marketing automation toolkit** that **provides 18 specialized agents, 93 commands, and 28 skills**. Unlike **Jasper** (AI writing only) or **HubSpot** (complex enterprise suite), we **integrate directly into your development workflow** because **modern marketers work in code, not clunky interfaces**.

### Mission Statement

> Democratize enterprise-grade marketing automation by making AI agents accessible to every team.

### Core Values

| Value | Definition | In Practice |
|-------|------------|-------------|
| **Open Source** | Free & accessible to all | MIT license, community-driven |
| **Developer-First** | Built for modern workflows | CLI-native, code-friendly |
| **Practical** | Real results over theory | Actionable outputs, not fluff |
| **Compounding** | Each use makes the next better | Templates, patterns, learning |

## 3. Tone by Context

### Documentation

- Clear, concise, scannable
- Code examples front and center
- Progressive disclosure (basics first)

**Example:**
```
## Quick Start

1. Install: `/plugin install agentkits-marketing`
2. Run: `/content:good "Write a tweet about [your product]"`
3. Done.
```

### Social Media

- Conversational, helpful
- Share tips and wins
- Engage with community

**Example:**
"Just shipped: 7 new CRO skills for landing pages, forms, and onboarding. Try /cro:page on your next project."

### Email

- Personal, value-first
- Short paragraphs
- Clear CTAs

### Error Messages

- Helpful, not apologetic
- Suggest solutions
- Link to docs

**Example:**
"Command not found. Did you mean /content:blog? See /help for all commands."

## 5. Writing Guidelines

### Headlines

| Do | Don't |
|----|-------|
| Specific benefits | Vague promises |
| Numbers when possible | Exaggeration |
| Action verbs | Passive voice |

**Examples:**
- "18 agents. 93 commands. Zero learning curve."
- "Write a blog post in 60 seconds"
- "CRO audit for your landing page"

### Body Copy

- Lead with value, not features
- One idea per paragraph
- Use code blocks for commands
- Break up text with lists

### CTAs

| Context | CTA Style |
|---------|-----------|
| Installation | "Get Started" |
| Documentation | "Learn More" |
| Community | "Join Us" |
| GitHub | "Star on GitHub" |

## 6. Terminology

### Use Consistently

| Term | Usage |
|------|-------|
| AgentKits | Product name (capital A, K) |
| agents | Lowercase, refers to AI agents |
| commands | Slash commands (/command:action) |
| skills | Domain knowledge bundles |
| toolkit | Not "platform" or "suite" |

### Avoid

| Instead Of | Use |
|------------|-----|
| "Revolutionary" | "Practical" |
| "AI-powered magic" | "AI marketing automation" |
| "Best-in-class" | Specific features |
| "Synergy" | Plain language |
| "Leverage" | "Use" |
