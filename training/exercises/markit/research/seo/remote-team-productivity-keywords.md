## 9. Technical SEO Considerations

### On-Page Optimization

| Element | Recommendation |
|---------|----------------|
| Title Tag | Include primary keyword + brand (60 chars max) |
| Meta Description | Include keyword, value prop, CTA (155 chars) |
| H1 | Exact match or close variant of target keyword |
| URL | Short, keyword-rich (e.g., /team-focus-time/) |
| Internal Links | Link to pillar from clusters, cross-link related |
| Schema | Article, FAQ, HowTo where appropriate |

### Page Structure

```
H1: Primary keyword
├── H2: Supporting topic (include secondary keyword)
│   ├── H3: Subtopic
│   └── H3: Subtopic
├── H2: Supporting topic
│   ├── H3: Subtopic
│   └── H3: Subtopic
└── H2: CTA section
```
